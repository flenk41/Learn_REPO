## 📖 Вывод

```rust
println!("Привет");                    // с переводом строки
print!("без переноса");                // без переноса
println!("{} + {} = {}", 2, 3, 2 + 3); // подстановка
println!("{name}", name = "Гена");     // именованные
eprintln!("это ошибка");               // вывод в поток ошибок (stderr)
```

| Макрос | Что делает |
|--------|-----------|
| `println!` | вывод + перенос строки |
| `print!` | вывод без переноса |
| `eprintln!` | вывод в stderr (для ошибок) |
| `format!` | собрать строку (не печатать) |

```rust
let s = format!("{}-{}", "a", "b");    // "a-b" (как f-строка, но в переменную)
```

## ⭐ Ввод с клавиатуры

Ввод в Rust строже, чем в других языках, — он заставляет обрабатывать возможные ошибки:

```rust
use std::io;                           // подключаем модуль ввода-вывода

fn main() {
    println!("Как тебя зовут?");

    let mut name = String::new();      // пустая изменяемая строка
    io::stdin()
        .read_line(&mut name)          // прочитать строку В name (по ссылке!)
        .expect("ошибка чтения");      // обработать возможную ошибку

    let name = name.trim();            // убрать перевод строки в конце
    println!("Привет, {}!", name);
}
```

🖼️ Разбор:
```
   io::stdin().read_line(&mut name)
                          └─ &mut = изменяемая ссылка: функция запишет В name
              .expect("...")  ← read_line вернул Result, expect разворачивает его
```

> ⚠️ `&mut name` — передаём **изменяемую ссылку**, чтобы `read_line` записал данные в нашу
> строку. Это снова намёк на заимствование (Уровень 2). `.expect()` обрабатывает ошибку:
> ввод может не сработать, и Rust заставляет это учесть.

### Считать число
```rust
let mut input = String::new();
io::stdin().read_line(&mut input).expect("ошибка");
let n: i32 = input.trim().parse().expect("это не число!");
println!("Удвоенное: {}", n * 2);
```

💡 `parse()` превращает строку в число. Тип (`i32`) подсказывает, во что парсить.
`.expect()` обработает случай «введено не число».

## 📖 Строки: первое знакомство

В Rust **два** строковых типа (полностью — в [Уровне 2](../02-memory/11-slices-strings.md)):

```rust
let literal: &str = "привет";          // строковый срез (&str) — неизменяемый, в коде
let owned: String = String::from("мир"); // String — владеющая, изменяемая, в куче
```

| | `&str` | `String` |
|--|--------|----------|
| Владение | заимствованный | владеющий |
| Изменяемость | нет | да (с `mut`) |
| Где | литералы, срезы | динамические строки |

Операции:
```rust
let mut s = String::from("привет");
s.push_str(", мир");                   // дописать
s.push('!');                           // добавить символ
println!("{}", s.len());               // длина в БАЙТАХ
println!("{}", s.to_uppercase());
let combined = String::from("a") + "b"; // склейка
```

> ⚠️ `len()` возвращает длину в **байтах**, а не в символах! Для Unicode (кириллица,
> эмодзи) символ может занимать несколько байт. Подробно — в Уровне 2.
