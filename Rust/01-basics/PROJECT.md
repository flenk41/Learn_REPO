# 🚀 Пет-проект уровня 1 — Угадай число

> 🎯 **Цель:** собрать всё из уровня 1 (ввод-вывод, `match`, циклы, парсинг, подключение
> внешней библиотеки) в классическую игру. Это первый проект из официальной книги Rust —
> идеальный старт.

---

## 📋 Что должна уметь программа

```
Угадай число! (от 1 до 100)
Твой вариант: 50
Слишком много!
Твой вариант: 25
Слишком мало!
Твой вариант: 37
🎉 Угадал за 3 попытки!
```

---

## ✅ Требования (минимум)

- [ ] Программа загадывает случайное число (библиотека `rand`).
- [ ] Цикл угадывания (`loop`), выход при правильном ответе (`break`).
- [ ] Подсказки «больше/меньше» через `match` по `Ordering`.
- [ ] Парсинг ввода (`parse`) с обработкой нечислового ввода (не падать).
- [ ] Счётчик попыток.

---

## 🪜 Пошаговый план

1. **Проект.** `cargo new guessing_game`.
2. **Зависимость.** Добавь в `Cargo.toml`: `rand = "0.8"`.
3. **Случайное число.** `rand::thread_rng().gen_range(1..=100)`.
4. **Цикл ввода.** Считывай строку, парси в число.
5. **Сравнение.** `match guess.cmp(&secret)` → `Ordering::Less/Greater/Equal`.
6. **Защита.** Нечисловой ввод — не падать, а просить заново (`match` на `parse`).
7. **Победа.** При `Equal` — поздравить, вывести попытки, `break`.

---

## 🧱 Скелет

```rust
use std::io;
use std::cmp::Ordering;
use rand::Rng;

fn main() {
    let secret = rand::thread_rng().gen_range(1..=100);
    let mut attempts = 0;
    println!("Угадай число! (от 1 до 100)");

    loop {
        println!("Твой вариант:");
        let mut input = String::new();
        io::stdin().read_line(&mut input).expect("ошибка чтения");

        // обработка нечислового ввода: parse возвращает Result
        let guess: u32 = match input.trim().parse() {
            Ok(num) => num,
            Err(_) => {
                println!("Введи число!");
                continue;
            }
        };

        attempts += 1;

        match guess.cmp(&secret) {
            Ordering::Less    => println!("Слишком мало!"),
            Ordering::Greater => println!("Слишком много!"),
            Ordering::Equal   => {
                println!("🎉 Угадал за {} попытки!", attempts);
                break;
            }
        }
    }
}
```
```powershell
cargo run
```

> 💡 Заметь: `match input.trim().parse()` обрабатывает `Ok`/`Err` — так Rust заставляет
> учесть ошибку парсинга. Это первое знакомство с `Result` (подробно — Уровень 3).
