# ✅ Задачи раздела · Углублённый Rust

> Продвинутые темы. Компилятор Rust — твой главный учитель: читай его ошибки внимательно.

---

## 🟢 Базовые

1. **Elision.** Функции, где лайфтаймы выводятся, и где аннотация обязательна. Объясни почему.
2. **Ассоциированный тип.** Реализуй свой `Iterator` (Counter/range) с `type Item`.
3. **Cell/RefCell.** Структура с внутренним счётчиком/кэшем, мутируемым через `&self`.

## 🟡 Средние

4. **dyn коллекция.** `Vec<Box<dyn Trait>>` с разными типами; обход. Зачем dyn vs generics?
5. **Rc<RefCell> граф.** Дерево/граф с общими изменяемыми узлами; реши цикл через `Weak`.
6. **Arc<Mutex>.** Раздели изменяемый счётчик между потоками без гонок.
7. **Tokio.** `tokio::spawn` нескольких задач, `join!`/`select!`. Конкурентность без потоков.

## 🔴 Со звёздочкой

8. ⭐ **static vs dyn.** Реализуй обработку через дженерик и через dyn; сравни (размер/скорость/гетерогенность).
9. ⭐ **Свой Future.** Реализуй простой Future (Ready/Pending→Ready); пойми poll/Waker.
10. ⭐ **Безопасная абстракция над unsafe.** Структура с unsafe внутри + безопасный API + `// SAFETY:`.
11. ⭐⭐ **Async-сервер.** TCP/echo на Tokio; сравни с event-loop из капстоуна.
