# 🔬 Раздел · Углублённый Rust

> Senior-уровень дал умные указатели, бесстрашную многопоточность, трейты-объекты, unsafe/FFI,
> async. Этот раздел — **глубокое погружение** в самые тонкие части Rust: лайфтаймы по-настоящему,
> продвинутые трейты, внутренняя изменяемость, как **на самом деле работает async**, и unsafe как
> инструмент построения безопасных абстракций.

Проходи после [Уровня 4 · Senior](../README.md) — нужны владение, трейты, smart pointers, async.

## 🎯 Зачем это

> **Здесь Rust раскрывается полностью.** Лайфтаймы перестают быть «борьбой с компилятором» и
> становятся инструментом; ты понимаешь, как async работает под капотом (а не «просто пишу
> async/await»), и как строить безопасные API над unsafe — то, на чём держится вся экосистема Rust.
