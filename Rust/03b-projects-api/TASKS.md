# ✅ Задачи · Проекты и API (Rust)

> Проверяй через `cargo clippy` и `cargo test`. Сложность: ⭐ · ⭐⭐ · ⭐⭐⭐

---

## 🟢 Блок A — структура проекта

1. ⭐ **Модуль в файле** — вынеси функции в `math.rs`, подключи через `mod`.
2. ⭐⭐ **Видимость** — pub и приватные элементы, поймай ошибку доступа к приватному.
3. ⭐⭐ **lib + main** — логика в `lib.rs`, использование из `main.rs`.
4. ⭐⭐ **Папка-модуль** `shapes/` с подмодулями.
5. ⭐⭐⭐ **Workspace** из крейта-библиотеки и крейта-программы.

---

## 🟢 Блок B — проектирование API

6. ⭐⭐ **Минимальный pub** — приватные поля, pub-методы доступа.
7. ⭐⭐ **Трейт-API** Storage с двумя реализациями.
8. ⭐⭐ **Свой тип ошибки** (enum + Display + Error), возврат через Result.
9. ⭐⭐⭐ **Builder** для структуры с опциональными полями.
10. ⭐⭐ **Doc-тесты** — `///` с примером, проверь через `cargo test`.

---

## 🟢 Блок C — веб-API

11. ⭐⭐ **GET через reqwest** (blocking) к публичному API.
12. ⭐⭐ **serde-структура** — распарси JSON в `struct` через `.json()`.
13. ⭐⭐ **Обработка** 404 и ошибок сети через `?`.
14. ⭐⭐ **Погода** через open-meteo в структуру.
15. ⭐⭐⭐ **Свой клиент** — `struct Client` с типизированными методами.

---

## 🧩 Где тренироваться

- [reqwest](https://docs.rs/reqwest), [serde](https://serde.rs) — документация.
