# 01 · Установка rustup и cargo 🛠️

> 🎯 **Цель блока:** установить Rust и проверить его работу. В Rust установка особенно
> простая — один инструмент `rustup` ставит всё.

---

## 📖 Что нам нужно

| Что | Зачем |
|-----|-------|
| **rustup** | установщик и менеджер версий Rust |
| **rustc** | компилятор (ставится через rustup) |
| **cargo** | менеджер пакетов и система сборки (ставится через rustup) |
| **VS Code** | редактор |

💡 В отличие от C/C++, где компилятор и инструменты ставятся отдельно, Rust даёт **всё
сразу** через `rustup`. Это одна из причин, почему с Rust приятно работать.

---

## 🪟 Установка на Windows (твоя система)

### Шаг 1. Установить rustup
1. Зайди на **https://rustup.rs** (официальный сайт).
2. Скачай **`rustup-init.exe`** и запусти.
3. В открывшемся окне терминала выбери **1) Proceed with standard installation**
   (нажми `1` и Enter, или просто Enter).

> ⚠️ rustup может попросить установить **Visual Studio C++ Build Tools** (нужны
> компоновщику Rust на Windows). Если попросит — согласись и установи их (или поставь
> заранее через Visual Studio Installer → «Разработка классических приложений на C++»).
> Без них линковка не сработает.

### Шаг 2. Проверить установку ✅
Открой **новый** PowerShell и введи:
```powershell
rustc --version
cargo --version
```
Должно появиться:
```
rustc 1.XX.0 (...)
cargo 1.XX.0 (...)
```
🎉 Rust установлен!

> ⚠️ Если команды не находятся — закрой и открой терминал заново (rustup добавляет себя в
> PATH, но текущее окно об этом не знает).

## 🍎 macOS / 🐧 Linux

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# следуй инструкциям, потом перезапусти терминал
rustc --version
cargo --version
```

---

## 💻 Установка редактора VS Code

1. Скачай с **https://code.visualstudio.com**, установи.
2. Открой Extensions (слева) → установи **rust-analyzer** (официальное расширение).

> 💡 **rust-analyzer** — потрясающий помощник: показывает типы прямо в коде, объясняет
> ошибки, автодополняет. Для Rust он почти обязателен — сильно ускоряет обучение.

---

## 📖 Полезные команды rustup

```powershell
rustup update              # обновить Rust до новой версии
rustup --version
rustup component add clippy    # линтер (ставится обычно сразу)
rustup component add rustfmt   # форматтер
rustup doc                 # открыть документацию офлайн
```

---

## 🗂️ Рабочая папка

Cargo сам создаёт структуру проектов (увидишь в следующем блоке), поэтому отдельную папку
для кода готовить не нужно — просто реши, где будут лежать проекты, например:
```
D:\LearnProject\my-code\Rust\
```
