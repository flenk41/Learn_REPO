⚠️ Запусти через локальный сервер

Браузер блокирует чтение файлов курса напрямую (через file://). Открой терминал в папке D:\LearnProject и выполни одну из команд:

python -m http.server 8000

или, если есть Node.js:

npx serve

Затем открой в браузере http://localhost:8000
(или просто запусти файл start-server.bat двойным кликом).
