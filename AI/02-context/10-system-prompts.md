# 10 · Системные промпты и роли 🖼️⭐

> 🎯 **Цель блока:** освоить системные промпты — постоянные инструкции, которые задают
> поведение ИИ на весь диалог. Это «настройка характера» помощника.

---

## 📖 Что такое системный промпт

**Системный промпт** — это инструкция, которая действует **на весь разговор** (а не на одно
сообщение). Он задаёт роль, правила, тон, формат — и ИИ следует им постоянно.

🖼️
```
   ┌─ Системный промпт (действует всегда) ─┐
   │ "Ты дружелюбный фитнес-тренер.         │
   │  Отвечай кратко, мотивирующе,          │
   │  всегда уточняй уровень подготовки."   │
   └────────────────────────────────────────┘
            ↓ применяется ко ВСЕМ сообщениям ниже
   Ты: "Хочу накачать пресс"  →  ИИ отвечает в заданном характере
   Ты: "А спину?"             →  ИИ всё ещё дружелюбный тренер
```

💡 Обычный промпт — разовая просьба. Системный промпт — **постоянная роль**. Он находится в
начале контекста и влияет на каждый ответ.

---

## ⭐ Где задаётся системный промпт

| Способ | Где |
|--------|-----|
| **Кастомные инструкции** | настройки ChatGPT/Claude («Custom Instructions», «Profile») |
| **Свои GPT / Projects** | ChatGPT GPTs, Claude Projects — сохранённая роль |
| **Через API** | поле `system` (Senior-уровень, модуль 22) |
| **Просто первое сообщение** | если нет спец-настроек — задай роль первым сообщением |

💡 Даже без специальных функций можно эмулировать системный промпт: сделай **первым
сообщением** подробную инструкцию о роли и правилах, и ИИ будет ей следовать весь диалог
(пока она в контексте).

---

## ⭐ Из чего состоит хороший системный промпт

```
РОЛЬ: Ты опытный редактор русскоязычных текстов.

ЗАДАЧА: Помогаешь улучшать тексты пользователя.

ПРАВИЛА:
- Сохраняй авторский смысл и стиль.
- Исправляй ошибки, но объясняй каждое серьёзное изменение.
- Предлагай, но не навязывай.

ТОН: профессиональный, доброжелательный, без снисходительности.

ФОРМАТ: сначала исправленный текст, затем список изменений.
```

| Элемент | Зачем |
|---------|-------|
| Роль | кем выступает ИИ |
| Задача | его основная функция |
| Правила | чёткие границы поведения |
| Тон | стиль общения |
| Формат | как оформлять ответы |

---

## ⭐ Зачем системные промпты

- **Постоянство.** Не нужно повторять инструкции в каждом сообщении.
- **Специализация.** Превращаешь универсальный ИИ в узкого эксперта (юрист, тренер, тьютор).
- **Контроль.** Задаёшь границы: что можно, что нельзя, как отвечать.
- **Экономия контекста.** Один раз настроил — дальше короткие запросы.

```
   Без системного промпта: каждый раз "ответь кратко, как эксперт по X, в формате Y"
   С системным промптом:   настроил один раз → просто задаёшь вопросы
```

## 📖 Примеры системных промптов

**Репетитор:**
```
Ты терпеливый репетитор по математике для школьников. Никогда не давай готовый
ответ сразу — веди ученика наводящими вопросами. Хвали за прогресс. Объясняй
простыми словами с примерами из жизни.
```

**Бизнес-ассистент:**
```
Ты деловой ассистент. Отвечай структурированно и по делу. Для любой задачи
сначала уточняй недостающие детали, затем предлагай решение. Тон — профессиональный,
краткий. Всегда предлагай следующий шаг.
```

**Партнёр по мозговому штурму:**
```
Ты креативный партнёр по брейнштормингу. Генерируй смелые, нестандартные идеи.
Не критикуй на этапе генерации. Давай по 5 идей за раз, от безопасных до безумных.
```

---

## ⚠️ Системный промпт ≠ абсолютная гарантия

> ⚠️ Системный промпт **сильно влияет** на поведение, но не делает его железобетонным. В
> очень длинном диалоге или при противоречивых запросах ИИ может отклоняться. Для критичных
> правил — повторяй их и проверяй вывод. (Про надёжность и «обход» правил — в Senior-блоке.)

## 🧪 Эксперимент: один вопрос — три системных промпта

Зайди в кастомные инструкции (или первым сообщением) и задай разные роли, потом задай
**один и тот же** вопрос:
1. «Ты строгий профессор».
2. «Ты весёлый друг».
3. «Ты краткий эксперт, отвечаешь одним предложением».

Сравни, как одна настройка меняет все ответы.

## ✅ Задачи

1. **Свой системный промпт.** Напиши системный промпт для помощника под свою частую задачу
   (роль + правила + тон + формат).
2. **Настрой кастомные инструкции** в своём ИИ, проверь, что роль держится несколько
   сообщений.
3. **Три роли.** Выполни эксперимент выше.
4. **Репетитор.** Сделай системный промпт «репетитор, который не даёт готовых ответов».
   Проверь, что ИИ ведёт тебя вопросами.
5. ⭐ **Эксперт-персона.** Создай детального ИИ-эксперта в своей профессии (роль, знания,
   стиль, границы). Это основа пет-проекта уровня.
