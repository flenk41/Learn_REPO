# 🚀 Пет-проект уровня 1 — Личный набор промптов

> 🎯 **Цель:** собрать собственную библиотеку проверенных промптов под свои реальные
> задачи. Это инструмент, которым ты будешь пользоваться каждый день.

---

## 📋 Что делаем

Личную «промпт-аптечку» — набор готовых шаблонов-промптов для частых задач, оформленных по
правилам Уровня 1 (роль + задача + контекст + формат + ограничения).

Пример карточки промпта:
```
### 📧 Деловое письмо
Роль: ты деловой ассистент.
Задача: напиши письмо [тип: просьба / отказ / благодарность].
Контекст: получатель — [кто], ситуация — [описание], цель — [чего хочу добиться].
Формат: 3 абзаца, с приветствием и подписью.
Ограничения: [тон: формальный/дружелюбный], не длиннее 150 слов.
```

---

## ✅ Требования (минимум)

- [ ] Минимум **8 промптов** под разные реальные задачи (работа, учёба, быт, творчество).
- [ ] Каждый промпт оформлен по структуре (хотя бы задача + контекст + формат).
- [ ] С полями-пропусками `[...]`, чтобы быстро адаптировать под конкретный случай.
- [ ] Каждый промпт **протестирован** на реальной задаче и доработан.
- [ ] Промпты сгруппированы по категориям.

---

## 🪜 Пошаговый план

1. **Собери задачи.** Выпиши 8–10 задач, где тебе реально помогает ИИ (письма, конспекты,
   идеи, переводы, планирование...).
2. **Напиши промпт-шаблон** для каждой по формуле Уровня 1.
3. **Протестируй** каждый на настоящем примере, доработай (итерации из модуля 07).
4. **Добавь пропуски** `[...]` для переменных частей.
5. **Сгруппируй** по категориям, оформи в файл/заметку.
6. **Отметь лучшие** — те, что используешь чаще всего.

---

## 💡 Идеи промптов для коллекции

| Категория | Промпты |
|-----------|---------|
| 📧 Работа | деловое письмо, ответ на жалобу, резюме встречи, описание вакансии |
| 📚 Учёба | объясни тему, составь конспект, проверь моё понимание, тест по теме |
| ✍️ Тексты | пост для соцсети, заголовки, редактура, сокращение |
| 🌍 Перевод | перевод с сохранением тона, локализация |
| 🧠 Мышление | плюсы/минусы решения, мозговой штурм, план задачи |
| 🏠 Быт | рецепт из продуктов, план поездки, список покупок |

---

## 🧱 Пример готового промпта из коллекции

```
### 📚 Объясни сложную тему
Ты терпеливый преподаватель.
Объясни мне тему: [ТЕМА].
Контекст: мой уровень — [новичок / средний], я уже знаю [что знаю].
Формат:
  1. Объяснение простыми словами (1 абзац)
  2. Аналогия из повседневной жизни
  3. Конкретный пример
  4. Один вопрос, чтобы проверить, понял ли я
Ограничения: без жаргона, дружелюбно.
```

---

## 💪 Усложнения (для «отлично»)

- ➕ 15+ промптов по всем сферам жизни.
- ➕ Для каждого — пример «до/после» (как улучшился результат благодаря структуре).
- ➕ Добавь промпты с few-shot примерами (из модуля 06).
- ➕ Версии промптов для разных ИИ, если пользуешься несколькими.
- ➕ Оформи как удобную шпаргалку (таблица/заметки с быстрым копированием).

---

## 🏆 Критерии готовности

| Уровень | Что сделано |
|---------|-------------|
| ✅ Зачёт | 8+ структурированных промптов, протестированы, по категориям |
| 🌟 Хорошо | 15+ промптов, с пропусками, с примерами до/после |
| 🏆 Отлично | библиотека на все случаи + few-shot промпты + удобное оформление |
