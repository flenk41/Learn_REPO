# 05 · Формат вывода

> 🎯 **Цель блока:** научиться задавать **форму** ответа. ИИ выдаёт ровно ту структуру,
> которую ты попросишь — это экономит время и делает результат готовым к использованию.

---

## 📖 Зачем управлять форматом

По умолчанию ИИ пишет «обычным текстом» — абзацами. Но часто нужно другое: список, таблица,
готовое письмо, код, JSON. Если не указать формат — получишь не то, что удобно.

```
   Без указания: ИИ пишет абзацами
   "...в виде таблицы": ИИ строит таблицу
   "...списком из 5 пунктов": ИИ даёт ровно 5 пунктов
```

💡 **Ты управляешь формой так же, как содержанием.** Просто скажи, как должен выглядеть ответ.

---

## ⭐ Главные форматы

```
"в виде маркированного списка"
"в виде нумерованного списка из 5 пунктов"
"в виде таблицы с колонками: Название | Плюсы | Минусы"
"одним абзацем"
"по шагам, каждый шаг с новой строки"
"в формате вопрос-ответ"
"в виде готового письма (с приветствием и подписью)"
"в формате JSON с полями name, price, category"
"кодом на Python с комментариями"
"в виде краткого резюме (1 предложение) + детали ниже"
```

🖼️
```
   "Сравни X и Y в виде таблицы"
        →  │ Критерий │  X  │  Y  │
           │  цена    │ ... │ ... │
           │  скорость│ ... │ ... │
```

---

## ⭐ Структура и заголовки

Для длинных ответов проси разбить на разделы:

```
"Структурируй ответ с заголовками:
 1. Краткий вывод
 2. Подробное объяснение
 3. Примеры
 4. Что делать дальше"
```

💡 Так ты получаешь не «стену текста», а удобный, навигируемый ответ. Особенно полезно для
обучения, отчётов, планов.

---

## 📖 Задавай длину и объём

```
"в 3 предложениях"
"не больше 100 слов"
"подробно, минимум 500 слов"
"кратко — только суть"
"TL;DR в начале, затем детали"
```

> 💡 ИИ примерно соблюдает длину, но не идеально считает слова. Если важна точность — проси
> «примерно N слов» и при необходимости уточняй «сократи вдвое» / «добавь деталей».

---

## ⭐ Шаблоны вывода — мощный приём

Дай ИИ **готовый шаблон** с пропусками — он заполнит:

```
Заполни этот шаблон для моего продукта (умные часы):

📦 Название: ___
💰 Цена: ___
✨ 3 главные фишки:
   1. ___
   2. ___
   3. ___
🎯 Для кого: ___
```

💡 Шаблон гарантирует, что ответ придёт в нужной структуре — ничего не забыто, ничего
лишнего. Незаменимо для повторяющихся задач (карточки товаров, резюме, анкеты).

---

## 📖 Формат для дальнейшего использования

Если ответ пойдёт в другую программу — проси машинный формат:

```
"Выдай список стран в формате CSV: страна,столица,население"
"Ответь только валидным JSON, без пояснений"
"Дай результат в Markdown, чтобы я вставил в документ"
```

> 💡 Уточнение «только JSON, без лишнего текста» важно: иначе ИИ может добавить «Вот ваш
> JSON:» — и формат сломается при автоматической обработке.

---

## 🧪 Эксперимент: один запрос — пять форматов

Возьми тему «преимущества удалённой работы» и попроси:
1. абзацем,
2. списком из 5 пунктов,
3. таблицей плюсы/минусы,
4. в формате твита,
5. как диалог двух людей.

Один контент — пять форм. Почувствуй контроль над выводом.

---

## ✅ Задачи

1. **Пять форматов.** Выполни эксперимент выше на своей теме.
2. **Таблица сравнения.** Попроси сравнить 3 варианта чего-либо в виде таблицы с твоими
   колонками.
3. **Шаблон.** Создай шаблон с пропусками для частой задачи, дай ИИ заполнить.
4. **Структура с заголовками.** Попроси разобрать сложную тему по разделам (вывод →
   объяснение → примеры → шаги).
5. **Машинный формат.** Попроси список данных строго в JSON или CSV, без пояснений.
