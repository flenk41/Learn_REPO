# 🚀 Финальные пет-проекты — стань экспертом по ИИ

> 🎯 **Цель:** выбери **один-два** серьёзных проекта, объединяющих все навыки: промптинг,
> контекст, продвинутые техники, RAG/агентов, оценку. Это твоё ИИ-портфолио.

---

## 🏆 Проект 1 — ИИ-ассистент для конкретной роли (рекомендую)

Полноценный специализированный ассистент под реальную задачу (твою работу, хобби, нишу).

**Что внутри:**
- 🎭 Детальный системный промпт (роль, правила, тон, границы) — модуль 10, 15.
- 🛡️ Защита от галлюцинаций, уточняющие вопросы — модуль 12, 11.
- 📋 Few-shot примеры идеальных ответов — модуль 14.
- ✅ Тестирование на наборе случаев — модуль 23.
- 📦 Оформлен как Custom GPT / Claude Project (если доступно).

**Результат:** работающий ассистент, которым реально пользуешься.

---

## 🏆 Проект 2 — Промпт-библиотека профессионала

Систематизированная коллекция проверенных промптов и рабочих процессов под целую сферу
(маркетинг, учёба, разработка, написание текстов).

**Что внутри:**
- 📚 20+ шаблонов с переменными по категориям.
- 🔧 Рабочие процессы (декомпозиция многошаговых задач) — модуль 16.
- 📊 Каждый промпт протестирован, с примерами до/после — модуль 23.
- 📖 Документация: когда что применять.

**Результат:** инструмент, экономящий часы и применимый каждый день.

---

## 🏆 Проект 3 — RAG-ассистент по своим материалам

ИИ, который отвечает на вопросы по твоей базе знаний (конспекты, документы, заметки).

**Что внутри:**
- 🗃️ База материалов + стратегия поиска (модуль 20).
- ⚓ Заземление: ответы только по данным, с источниками — модуль 12.
- 🔍 Без программирования: «ручной RAG» / чат с документами.
- 💻 С программированием: настоящий RAG через API + векторную БД (связь с языковыми треками).

**Результат:** персональный «эксперт» по твоим материалам.

---

## 🏆 Проект 4 — Автоматизация рабочего процесса

Сквозная автоматизация реальной рутины (отчёты, контент, обработка данных, поддержка).

**Что внутри:**
- ⚙️ Многошаговый процесс с шаблонами и проверками — модуль 16, 23.
- 🤖 Возможно агент/инструменты — модуль 21.
- 💻 Возможно через API для пакетной обработки — модуль 22.
- 📈 Замер экономии времени (было/стало).

**Результат:** процесс, встроенный в твою работу.

---

## 🏆 Проект 5 — ИИ-инструмент на коде (для программистов)

Свяжи курс по ИИ с языковыми треками: построй программу, использующую ИИ через API.

**Идеи:**
- 🌐 чат-бот / помощник на Python (или Rust/C++ через API);
- 📝 пакетный обработчик текстов (классификация, перевод, резюме);
- 🔌 RAG-система с векторной БД;
- 🤖 простой агент с инструментами.

**Что внутри:** API-интеграция (модуль 22), обработка ошибок, безопасное хранение ключей,
тесты. Это объединяет ИИ-курс с разделами «Проекты и API» языковых треков.

## 📋 Требования к ЛЮБОМУ финальному проекту

- [ ] Решает **реальную** задачу (твою, не учебную абстракцию).
- [ ] Использует минимум **5 техник** из курса.
- [ ] Есть **проверки качества** (тесты, защита от галлюцинаций).
- [ ] **Протестирован** на реальных случаях.
- [ ] **Документирован**: как пользоваться, примеры.
- [ ] Соблюдает **этику и безопасность** (модуль 24).

## 🎓 Что дальше

Ты прошёл путь от первого промпта до экспертной работы с ИИ. Теперь ты:
- пишешь сильные промпты системно;
- понимаешь, как ИИ устроен (контекст = память, токены, галлюцинации);
- владеешь продвинутыми техниками (chain-of-thought, few-shot, декомпозиция, RAG, агенты);
- знаешь границы и используешь ИИ ответственно.

**Связь со всем курсом:** во всех языках ядро — память; у ИИ ядро — контекст (его память).
Понимание памяти на всех уровнях — от байтов в C до контекстного окна ИИ — делает тебя
сильным и разносторонним специалистом.

Куда двигаться:
- **Глубже в ИИ:** построение ИИ-продуктов, RAG/агенты на коде, fine-tuning.
- **Свяжи с программированием:** ИИ-инструменты на Python/Rust через API.
- **Применяй ежедневно:** лучший рост — реальное использование в работе и жизни.
