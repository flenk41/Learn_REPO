## 🛠️ Как начать (на примере любого чат-ИИ)

1. Зайди на сайт выбранного ИИ.
2. Зарегистрируйся (email или вход через Google).
3. Увидишь поле ввода — это и есть место, куда пишут промпты.
4. Напиши что-нибудь простое: `Привет! Объясни, что ты умеешь.`
5. Нажми отправить — получишь ответ.

🖼️ Интерфейс чат-ИИ устроен одинаково:
```
   ┌────────────────────────────────────────┐
   │  История сообщений (диалог)            │
   │   Ты: Привет!                          │
   │   ИИ: Здравствуйте! Чем помочь?        │
   │                                        │
   ├────────────────────────────────────────┤
   │  [ Поле ввода промпта        ]  [▶]    │
   └────────────────────────────────────────┘
```

---

## 📖 Полезные элементы интерфейса

| Элемент | Что делает |
|---------|-----------|
| **Новый чат** | начать с чистого контекста (модель «забывает» прошлое) |
| **История чатов** | вернуться к прошлым разговорам |
| **Выбор модели** | переключить версию (быстрая ↔ мощная) |
| **Регенерация** | попросить переписать ответ заново |
| **Редактирование** | изменить свой промпт и отправить снова |
| **Загрузка файлов** | дать ИИ документ/картинку (в продвинутых версиях) |

> 💡 **Кнопка «Новый чат»** важна для понимания контекста: новый чат = чистая память.
> Если хочешь, чтобы ИИ «не путался» в темах — начинай новый чат для новой задачи.

---

## 📖 Бесплатные vs платные: что важно

- **Лимиты.** В бесплатных версиях бывает ограничение на число сообщений в час/день.
- **Модель.** Платные дают доступ к самым мощным моделям (лучше рассуждают, длиннее контекст).
- **Функции.** Загрузка файлов, генерация картинок, работа с интернетом — иногда только в
  платных.

💡 Для прохождения этого курса бесплатной версии **достаточно** до Senior-уровня. Платная
пригодится для серьёзной ежедневной работы.

## 🛠️ Совет: заведи «черновик промптов»

Создай простой текстовый файл (или заметку), куда будешь складывать удачные промпты. По мере
курса ты соберёшь личную библиотеку — это твой будущий пет-проект Уровня 1.

```
Мои промпты/
├── работа/       (письма, резюме, отчёты)
├── учёба/        (объяснения, конспекты)
└── творчество/   (тексты, идеи)
```

## ✅ Задачи

1. **Регистрация.** Заведи аккаунт хотя бы в одном чат-ИИ.
2. **Первый контакт.** Спроси у ИИ, что он умеет, какая у него модель и до какой даты он
   знает информацию.
3. **Сравнение.** Если есть доступ к двум ИИ — задай обоим один вопрос, сравни ответы.
4. **Новый чат.** Напиши что-то, начни новый чат, спроси «о чём мы только что говорили?» —
   убедись, что ИИ не помнит (чистый контекст).
5. **Черновик.** Создай файл/заметку для будущей коллекции промптов.
