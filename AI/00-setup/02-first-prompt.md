# 02 · Первый промпт 🖼️

> 🎯 **Цель блока:** написать первые промпты и на собственном опыте увидеть, как
> формулировка меняет ответ. Это фундамент всего курса.

---

## 📖 Что такое промпт

**Промпт** (prompt) — это твой запрос к ИИ: вопрос, задание, инструкция. Всё, что ты
пишешь в поле ввода.

```
   Простой промпт:   "Что такое фотосинтез?"
   Промпт-задание:   "Напиши письмо коллеге с просьбой перенести встречу"
   Промпт-инструкция: "Исправь ошибки в этом тексте: ..."
```

🎯 Качество ответа ИИ **напрямую** зависит от качества промпта. Тот же ИИ на плохой промпт
даст плохой ответ, на хороший — отличный. Промпт — это твой главный инструмент.

---

## ⭐ Эксперимент: плохой промпт vs хороший

Попробуй сам два промпта и сравни ответы.

**❌ Расплывчатый промпт:**
```
Напиши про собак
```
ИИ выдаст что-то общее и непонятно для чего — он не знает, что тебе нужно.

**✅ Конкретный промпт:**
```
Напиши короткий (3 абзаца) дружелюбный текст для детей 7 лет о том,
почему собаки — хорошие друзья. Простыми словами, с примерами.
```
ИИ выдаст именно то, что нужно — потому что ты дал **цель, объём, аудиторию, тон**.

🖼️
```
   Расплывчатый промпт  →  расплывчатый ответ  (ИИ угадывает, чего ты хочешь)
   Конкретный промпт    →  точный ответ        (ИИ знает, что делать)
```

💡 **Главный вывод первого дня:** ИИ не читает мысли. Чем больше полезных деталей в
промпте — тем лучше результат. Это мы будем развивать весь курс.

---

## 📖 Анатомия запроса (превью)

Хороший промпт часто содержит:
- **Что сделать** (задача): «напиши», «объясни», «исправь», «придумай».
- **О чём** (тема/данные).
- **Как** (формат, объём, тон, аудитория).

```
   "Объясни [что: квантовую физику] [как: простыми словами, для подростка,
    в 5 предложениях]"
```

Полную анатомию разберём в [модуле 03](../01-basics/03-prompt-anatomy.md). Пока просто
запомни: **добавляй детали**.

---

## 📖 Базовые типы задач для ИИ

Попробуй каждый — это «разминка»:

```
1. Объяснение:   "Объясни простыми словами, что такое инфляция"
2. Создание:     "Придумай 5 названий для кофейни в морском стиле"
3. Редактура:    "Сделай этот текст вежливее: [текст]"
4. Суммирование: "Перескажи этот текст в 3 пунктах: [текст]"
5. Перевод:      "Переведи на английский официальным тоном: [текст]"
6. Анализ:       "Какие плюсы и минусы у этого решения: [решение]"
```

💡 Большинство реальных задач — это комбинация этих типов. Освоив их, ты уже сможешь решать
кучу повседневных дел с ИИ.

---

## ⚠️ Частые ошибки новичка

```
❌ Слишком коротко:  "помоги с резюме"  (с чем именно? какая профессия?)
❌ Всё сразу:        огромный запрос с 10 разными задачами в куче
❌ Без контекста:    "это нормально?" (что «это»? ИИ не видит твою ситуацию)
❌ Ожидание чтения мыслей: "сделай красиво" (что для тебя «красиво»?)
```

✅ **Лечение:** будь конкретным, давай контекст, по одной задаче за раз. Этому посвящён весь
Уровень 1.

## ✅ Задачи

1. **Плохой vs хороший.** Возьми любую тему, напиши про неё расплывчатый промпт и
   конкретный. Сравни ответы. Запиши вывод.
2. **Шесть типов.** Попробуй каждый из 6 типов задач выше на своей теме.
3. **Добавь деталей.** Возьми промпт «напиши пост» и постепенно добавляй детали (тема →
   аудитория → тон → объём), отправляя после каждого добавления. Смотри, как меняется ответ.
4. **Твоя реальная задача.** Придумай задачу из своей жизни/работы и реши её с ИИ. Если ответ
   не идеален — подумай, каких деталей не хватило в промпте.

4. Назови 4 базовых типа задач для ИИ.
5. Назови 2 частые ошибки новичка.
