# 🚀 Пет-проект уровня 2 — Свой динамический массив (Vector)

> 🎯 **Цель:** реализовать «растущий массив» (как `std::vector` в C++ или `list` в Python),
> который сам увеличивается при добавлении элементов. Это **квинтэссенция** работы с
> памятью: malloc, realloc, free, указатели, владение памятью.

> 💪 Это серьёзный проект. Сделав его, ты по-настоящему поймёшь, как изнутри устроены
> динамические структуры данных во всех языках.

---

## 📖 Что такое динамический массив

Обычный массив имеет фиксированный размер. Vector — это массив, который:
- знает свою **длину** (`size` — сколько элементов реально лежит);
- знает свою **вместимость** (`capacity` — сколько влезет без перевыделения);
- при заполнении **сам увеличивается** (обычно удваивает capacity через `realloc`).

🖼️ Как растёт вектор:

```
push 1: size=1 cap=1   [1]
push 2: size=2 cap=2   [1][2]            ← realloc до 2
push 3: size=3 cap=4   [1][2][3][_]      ← realloc до 4 (удвоили)
push 4: size=4 cap=4   [1][2][3][4]
push 5: size=5 cap=8   [1][2][3][4][5][_][_][_]  ← realloc до 8
```

💡 Удвоение capacity, а не +1 каждый раз — чтобы не вызывать дорогой `realloc` на
каждом добавлении.

## 📋 API, которое нужно реализовать

```c
typedef struct {
    int   *data;       // указатель на динамический массив в куче
    size_t size;       // сколько элементов сейчас
    size_t capacity;   // сколько влезет без перевыделения
} Vector;

void vec_init(Vector *v);                 // инициализация (пустой вектор)
void vec_push(Vector *v, int value);      // добавить в конец (растёт при необходимости)
int  vec_get(const Vector *v, size_t i);  // получить элемент по индексу
void vec_set(Vector *v, size_t i, int value);  // изменить элемент
int  vec_pop(Vector *v);                  // удалить и вернуть последний
size_t vec_size(const Vector *v);         // текущая длина
void vec_free(Vector *v);                 // освободить ВСЮ память
```

## 🪜 Пошаговый план

1. **Структура.** Опиши `struct Vector`. Напиши `vec_init` (data = NULL, size = 0, cap = 0).
2. **Push без роста.** Сначала пусть `vec_init` выделяет capacity = 4. Реализуй `vec_push`
   без проверки переполнения (пока влезает).
3. **Рост.** В `vec_push` добавь: если `size == capacity` → удвой capacity через `realloc`
   (если capacity == 0, начни с 1).
4. **Доступ.** Реализуй `vec_get` / `vec_set` с проверкой границ.
5. **Pop.** Реализуй `vec_pop` (уменьшает size, возвращает последний элемент).
6. **Free.** `vec_free`: `free(data)`, обнули поля.
7. **Тесты.** В `main` напиши проверки: добавь 100 элементов, выведи, поменяй, удали.
8. **ASan.** Прогони под `-fsanitize=address`. Добейся чистого вывода.

## 🧱 Заготовка-скелет

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct {
    int   *data;
    size_t size;
    size_t capacity;
} Vector;

void vec_init(Vector *v) {
    v->data = NULL;
    v->size = 0;
    v->capacity = 0;
}

void vec_push(Vector *v, int value) {
    if (v->size == v->capacity) {
        size_t new_cap = (v->capacity == 0) ? 1 : v->capacity * 2;
        int *tmp = realloc(v->data, new_cap * sizeof(int));
        if (tmp == NULL) {
            fprintf(stderr, "Out of memory\n");
            exit(1);
        }
        v->data = tmp;
        v->capacity = new_cap;
    }
    v->data[v->size] = value;
    v->size++;
}

// vec_get, vec_set, vec_pop, vec_size — реализуй сам!

void vec_free(Vector *v) {
    free(v->data);
    v->data = NULL;
    v->size = 0;
    v->capacity = 0;
}

int main(void) {
    Vector v;
    vec_init(&v);

    for (int i = 0; i < 10; i++)
        vec_push(&v, i * i);

    for (size_t i = 0; i < v.size; i++)
        printf("%d ", v.data[i]);
    printf("\n(size=%zu, capacity=%zu)\n", v.size, v.capacity);

    vec_free(&v);
    return 0;
}
```

Скомпилируй и проверь:
```bash
gcc -Wall -Wextra -fsanitize=address -g vector.c -o vector
./vector
```

## 💪 Усложнения (для отметки «отлично»)

- ➕ `vec_insert(v, index, value)` — вставка в середину (сдвиг элементов).
- ➕ `vec_remove(v, index)` — удаление из середины.
- ➕ `vec_shrink(v)` — уменьшить capacity до size (вернуть лишнюю память).
- ➕ Сделай вектор **обобщённым** через `void*` и `sizeof` элемента (хранить любой тип) —
