# ✅ Задачи · Уровень 2 (ПАМЯТЬ)

> ⚠️ **Правило этого уровня:** компилируй **каждое** решение с
> `gcc -Wall -Wextra -fsanitize=address -g`. Решение засчитано, только если ASan молчит
> (нет утечек и ошибок памяти).

Сложность: ⭐ легко · ⭐⭐ средне · ⭐⭐⭐ сложно

---

## 🟢 Блок A — указатели

1. ⭐ **Три значения.** Через указатели напиши функцию, возвращающую сумму, разность и
   произведение двух чисел сразу.
2. ⭐ **swap всего.** Реализуй `swap` для `int`, для `double`, для `char`. Сравни.
3. ⭐⭐ **Сортировка пузырьком.** Отсортируй массив, используя функцию `swap` через указатели.
4. ⭐⭐ **min/max за один проход.** Функция находит минимум и максимум массива, возвращая
   оба через указатели.
5. ⭐⭐ **Указатель на середину.** Функция принимает массив и размер, возвращает указатель
   на средний элемент.

---

## 🟢 Блок B — строки (свои реализации)

> Реализуй **без** `<string.h>` — это лучший способ понять строки.

6. ⭐ `my_strlen` — длина строки.
7. ⭐ `my_strcpy` — копирование строки.
8. ⭐⭐ `my_strcmp` — сравнение строк.
9. ⭐⭐ `my_strcat` — склейка строк.
10. ⭐⭐ `my_strrev` — реверс строки на месте.
11. ⭐⭐ **Подсчёт символов.** Сколько раз символ встречается в строке.
12. ⭐⭐⭐ `my_strstr` — поиск подстроки в строке.
13. ⭐⭐⭐ **Удаление пробелов.** Убери лишние пробелы из строки на месте.

---

## 🟢 Блок C — динамическая память

14. ⭐ **Дубликат массива.** Верни новый динамический массив-копию.
15. ⭐⭐ **Слияние массивов.** Из двух массивов сделай один новый (динамический).
16. ⭐⭐ **Фильтр.** Из массива чисел верни новый массив только из чётных (размер заранее
    неизвестен — посчитай или используй realloc).
17. ⭐⭐ **Своя getline.** Считай строку неизвестной длины посимвольно, расширяя буфер через
    `realloc`.
18. ⭐⭐⭐ **Разбиение строки (split).** Раздели строку по пробелам на массив слов
    (динамический массив динамических строк). Освободи всё корректно.
19. ⭐⭐⭐ **Динамическая матрица.** Создай матрицу `n×m`, транспонируй её в новую,
    корректно освободи обе.

## 🟢 Блок D — структуры данных на указателях (подготовка к Middle)

20. ⭐⭐ **Стек на массиве.** Реализуй стек (push/pop/peek) на динамическом массиве.
21. ⭐⭐⭐ **Динамический массив (vector).** Это твой пет-проект ниже — начни здесь.
22. ⭐⭐⭐ **Односвязный список.** Узел `{ int value; struct Node *next; }`. Реализуй
    добавление в начало, в конец, печать, удаление, освобождение всего списка.
