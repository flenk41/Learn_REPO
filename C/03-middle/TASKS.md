# ✅ Задачи · Уровень 3 (Middle)

> Компилируй с `gcc -Wall -Wextra -fsanitize=address -g`. Всё, что выделяет память,
> обязано её освобождать — ASan должен молчать.

Сложность: ⭐ легко · ⭐⭐ средне · ⭐⭐⭐ сложно

---

## 🟢 Блок A — структуры

1. ⭐ **Геометрия.** Структуры `Point`, `Circle`, `Rect`. Функции площади, периметра,
   пересечения, попадания точки.
2. ⭐⭐ **Комплексные числа.** Структура `Complex`, операции +, −, ×, модуль.
3. ⭐⭐ **Каталог товаров.** Массив структур `Product`. Поиск, сортировка по цене,
   подсчёт суммарной стоимости склада.
4. ⭐⭐ **Время.** Структура `Time { int h, m, s; }`. Сложение времени, разница, нормализация.

---

## 🟢 Блок B — структуры данных

5. ⭐⭐ **Связный список.** Полный набор: вставка в начало/конец, удаление, поиск, разворот,
   корректное освобождение.
6. ⭐⭐ **Стек — проверка скобок.** Используя стек, проверь правильность скобок `()[]{}`
   в строке.
7. ⭐⭐ **Очередь.** Реализуй очередь, смоделируй обслуживание клиентов.
8. ⭐⭐⭐ **Дерево поиска.** Вставка, поиск, in-order обход (сортировка), высота, удаление дерева.
9. ⭐⭐⭐ **Хеш-таблица (основы).** Массив списков (chaining), функции put/get/remove
   для строковых ключей.

---

## 🟢 Блок C — указатели на функции

10. ⭐⭐ **map/filter/reduce.** Реализуй три классические функции для массива int.
11. ⭐⭐ **Сортировка через qsort.** Отсортируй массив структур по разным полям.
12. ⭐⭐⭐ **Мини-CLI.** Таблица команд `{ имя, обработчик }`, цикл чтения команд и вызов.

---

## 🟢 Блок D — файлы и модульность

13. ⭐ **wc.** Посчитай строки/слова/символы в файле.
14. ⭐⭐ **CSV → структуры.** Распарси CSV-файл в массив структур, выведи таблицей.
15. ⭐⭐ **Журнал (лог).** Дописывай события с временной меткой в файл.
16. ⭐⭐⭐ **Многофайловый проект.** Раздели любой свой крупный проект на `.h`/`.c` + Makefile.

---

## 🟢 Блок E — биты

17. ⭐ **Двоичное представление** числа + подсчёт единиц.
18. ⭐⭐ **Система прав** через флаги (выдать/отнять/проверить).
19. ⭐⭐ **XOR-шифр** строки с ключом.
20. ⭐⭐⭐ **Упаковка RGB/даты** в одно число и распаковка.
