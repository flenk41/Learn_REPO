## 📂 Подробнее о модулях

### 13 · Структуры, union, enum
- `struct` — объединение разных данных в один тип
- 🖼️ как поля структуры лежат в памяти (подряд, с выравниванием)
- доступ через `.` и `->` (для указателей на структуру)
- `typedef` — удобные имена типов
- `union` — несколько полей в одной памяти
- `enum` — именованные константы
- 🚀 проект: каталог книг / база контактов

### 14 · Структуры данных на указателях
- односвязный и двусвязный список 🖼️ (узлы в куче, связанные указателями)
- стек и очередь
- бинарное дерево поиска
- хеш-таблица (основы)
- 🚀 проект: библиотека связного списка / телефонная книга

### 15 · Указатели на функции
- функция как данные, `int (*fp)(int, int)`
- callback-и, таблицы функций
- 🚀 проект: калькулятор на таблице функций / простой обработчик команд

### 16 · Препроцессор и модульность
- макросы, `#define`, `#ifdef`, include guards
- разделение на `.h` и `.c`
- сборка через `make` / `Makefile`
- 🖼️ как линковщик собирает несколько файлов
- 🚀 проект: своя мини-библиотека из нескольких файлов

### 17 · Работа с файлами
- `fopen`, `fread`, `fwrite`, `fclose`
- текстовые и бинарные файлы
- 🖼️ буферизация ввода-вывода
- 🚀 проект: парсер CSV / простой текстовый редактор / лог-анализатор

### 18 · Битовые операции
- `& | ^ ~ << >>`
- битовые маски, флаги
- битовые поля в структурах
- 🚀 проект: упаковка данных / простой шифр / работа с цветом RGB
