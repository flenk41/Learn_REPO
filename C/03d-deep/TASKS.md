# ✅ Задачи раздела · Углублённый C

> Продвинутый системный C. Используй godbolt (видеть оптимизатор), санитайзеры (UBSan/ASan),
> `gcc -E` (раскрытие макросов).

---

## 🟢 Базовые

1. **UB исчезает.** Код с выходом за границы после проверки; сравни -O0 и -O2 — исчезла ли проверка?
2. **void* стек.** Обобщённый стек на `void*`. Объясни потерю типобезопасности.
3. **# и ##.** `LOG(x)` с именем переменной; генерация имён через `##`.

## 🟡 Средние

4. **Strict aliasing.** Реинтерпретируй float как int через каст указателя (UB) и memcpy. Поведение на -O2?
5. **Макро-Vec.** `DEFINE_VEC(T, name)` (push/get/grow) для int и своей структуры.
6. **X-macros.** Из одного списка: enum + массив строк + `to_string`.
7. **container_of + intrusive.** Реализуй container_of; intrusive-список, объект в двух списках.

## 🔴 Со звёздочкой

8. ⭐ **UBSan-охота.** Прогони разные UB под `-fsanitize=undefined`. Что ловит каждый?
9. ⭐ **volatile/атомики.** Покажи выброс чтения без volatile (-O2); реализуй потокобезопасный счётчик
   через `_Atomic`. Объясни, почему volatile не годится для потоков.
10. ⭐ **Tagged union.** Тип-значение (int/float/string) с тегом; безопасная обработка (мостик к интерпретатору).
11. ⭐⭐ **Flexible array + один malloc.** Структура «строка/пакет» с заголовком и данными в одной аллокации.
