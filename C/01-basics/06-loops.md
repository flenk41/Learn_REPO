# 06 · Циклы

## 📖 Зачем циклы

Представь, что надо вывести числа от 1 до 100. Писать 100 раз `printf`? Нет.
Цикл повторяет блок кода, пока выполняется условие.

---

## 🔁 Цикл `while` — «пока условие истинно»

```c
int i = 1;
while (i <= 5) {
    printf("%d ", i);
    i = i + 1;          // ⚠️ без этого — бесконечный цикл!
}
// Вывод: 1 2 3 4 5
```

🖼️ Как крутится цикл:

```
   i=1
    │
    ▼
 ┌─────────────────┐
 │ i <= 5 ?        │──нет──► выход из цикла
 └────────┬────────┘
       да │
          ▼
   печать i, затем i = i + 1
          │
          └──────► обратно к проверке
```

> ⚠️ **Бесконечный цикл** — если забыть менять переменную условия (`i = i + 1`),
> цикл крутится вечно, программа зависает. Останови её: **Ctrl + C** в терминале.

## 🔁 Цикл `for` — когда знаешь, сколько раз

Самый частый цикл. Три части в одной строке:

```c
for (int i = 1; i <= 5; i++) {
    printf("%d ", i);
}
//    └──┬───┘ └──┬──┘ └┬┘
//   старт    условие  шаг
```

```
for ( ИНИЦИАЛИЗАЦИЯ ; УСЛОВИЕ ; ШАГ )
       (1 раз)       (каждый    (после
                      раз)       каждого
                                 прохода)
```

`i++` — это короткая запись `i = i + 1` (увеличить на 1).

### Аналоги инкремента/декремента

```c
i++;        // i = i + 1
i--;        // i = i - 1
i += 3;     // i = i + 3
i *= 2;     // i = i * 2
```

## 🔁 Цикл `do-while` — выполнить хотя бы раз

Проверка условия — **в конце**, поэтому тело выполнится минимум один раз.

```c
int n;
do {
    printf("Введи положительное число: ");
    scanf("%d", &n);
} while (n <= 0);       // повторяем, пока ввод неверный
```

💡 Идеально для меню и проверки ввода.

## 🎛️ Управление циклом: `break` и `continue`

```c
for (int i = 1; i <= 10; i++) {
    if (i == 5) break;      // полностью ВЫЙТИ из цикла
    printf("%d ", i);
}
// Вывод: 1 2 3 4

for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) continue;   // ПРОПУСТИТЬ остаток шага, перейти к следующему i
    printf("%d ", i);
}
// Вывод: 1 3 5 7 9
```

| Команда | Действие |
|---------|----------|
| `break` | выйти из цикла полностью |
| `continue` | пропустить текущую итерацию, перейти к следующей |

## 🔁 Вложенные циклы

Цикл внутри цикла — например, для таблиц и двумерных структур:

```c
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        printf("%d*%d=%d  ", i, j, i * j);
    }
    printf("\n");
}
```

🖼️ Внутренний цикл проходит **полностью** на каждый шаг внешнего:

```
i=1: j=1, j=2, j=3   ↵
i=2: j=1, j=2, j=3   ↵
i=3: j=1, j=2, j=3   ↵
```

## ✅ Задачи

1. **Счёт.** Выведи числа от 1 до 20 в одну строку.
2. **Сумма.** Посчитай сумму чисел от 1 до 100 (должно получиться 5050).
3. **Факториал.** Считай `n`, посчитай `n!` (1*2*3*...*n).
4. **Таблица умножения.** Выведи таблицу 9×9 с помощью вложенных циклов.
5. **Реверс числа.** Считай число и выведи его задом наперёд (123 → 321). *(Подсказка:
   `% 10` даёт последнюю цифру, `/ 10` убирает её.)*
6. **Простое число?** Считай число и определи, простое ли оно.
7. **Угадай число.** Загадай число в коде (например 42). Пользователь вводит варианты,
   программа говорит «больше»/«меньше», пока не угадает. Считай количество попыток.
8. **Ёлочка.** Считай высоту `n`, нарисуй пирамиду из звёздочек высотой `n` (вложенные циклы).
