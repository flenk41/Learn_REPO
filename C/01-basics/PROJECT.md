# 🚀 Пет-проект уровня 1 — Консольный калькулятор

> 🎯 **Цель:** собрать всё из уровня 1 (ввод-вывод, условия, циклы, функции) в одну
> законченную программу. Это твой первый «настоящий» проект.

---

## 📋 Что должна уметь программа

Консольный калькулятор, который работает в цикле, пока пользователь не выйдет:

```
==== КАЛЬКУЛЯТОР ====
1. Сложение
2. Вычитание
3. Умножение
4. Деление
5. Остаток
0. Выход
Выбери операцию: 1
Введи два числа: 10 5
10 + 5 = 15

Выбери операцию: 4
Введи два числа: 10 0
Ошибка: деление на ноль!

Выбери операцию: 0
Пока!
```

---

## ✅ Требования (минимум)

- [ ] Меню в цикле (`do-while` или `while`), выход по `0`.
- [ ] Минимум 5 операций: `+ - * / %`.
- [ ] Каждая операция — **отдельная функция** (`add`, `subtract`, и т.д.).
- [ ] Обработка **деления на ноль** (не должно падать).
- [ ] Меню выбора через `switch`.
- [ ] Компилируется без предупреждений с `-Wall -Wextra`.

---

## 🪜 Пошаговый план (если не знаешь, с чего начать)

1. **Каркас.** Сделай `main` с бесконечным циклом, который печатает меню и считывает выбор.
2. **Выход.** Добавь обработку `0` → `break`.
3. **Одна операция.** Реализуй только сложение: считай два числа, выведи сумму.
4. **Функции.** Вынеси сложение в `double add(double a, double b)`. Проверь, что работает.
5. **Остальные операции.** Добавь `subtract`, `multiply`, остальные по аналогии.
6. **switch.** Свяжи пункты меню с функциями через `switch`.
7. **Защита.** Добавь проверку деления на ноль и проверку, что введено именно число
   (`scanf` вернул 1).
8. **Полировка.** Красивый вывод, понятные сообщения.

---

## 💪 Усложнения (по желанию — для отметки «отлично»)

- ➕ Возведение в степень и квадратный корень (`#include <math.h>`, компилируй с `-lm`).
- ➕ Память калькулятора: команды `M+`, `MR`, `MC` (хранить результат в переменной).
- ➕ Счётчик выполненных операций, вывод при выходе.
- ➕ Работа с историей: показывать последние 5 вычислений.
- ➕ Меню на цифры **и** буквы (`+`, `-`) — обрабатывай оба варианта.

## 🧱 Заготовка-скелет

Не копируй слепо — пойми каждую строку и допиши сам:

```c
#include <stdio.h>

double add(double a, double b)      { return a + b; }
double subtract(double a, double b) { return a - b; }
double multiply(double a, double b) { return a * b; }
// divide реализуй сам, не забудь про деление на 0

void print_menu(void) {
    printf("\n==== КАЛЬКУЛЯТОР ====\n");
    printf("1. Сложение\n2. Вычитание\n3. Умножение\n");
    printf("4. Деление\n0. Выход\n");
    printf("Выбери операцию: ");
}

int main(void) {
    int choice;
    double a, b;

    do {
        print_menu();
        if (scanf("%d", &choice) != 1) {
            printf("Введи число!\n");
            // тут нужно очистить ввод — подумай как
            break;
        }

        if (choice == 0) {
            printf("Пока!\n");
            break;
        }

        printf("Введи два числа: ");
        scanf("%lf %lf", &a, &b);

        switch (choice) {
            case 1: printf("= %.2f\n", add(a, b)); break;
            case 2: printf("= %.2f\n", subtract(a, b)); break;
            // дополни остальные
            default: printf("Нет такой операции\n");
        }
    } while (1);

    return 0;
}
```
