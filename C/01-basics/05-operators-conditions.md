# 05 · Операторы и условия

> 🎯 **Цель блока:** научиться считать выражения и принимать решения в коде
> (`if`/`else`/`switch`).

---

## 📖 Арифметические операторы

| Оператор | Значение | Пример | Результат |
|----------|----------|--------|-----------|
| `+` | сложение | `5 + 3` | 8 |
| `-` | вычитание | `5 - 3` | 2 |
| `*` | умножение | `5 * 3` | 15 |
| `/` | деление | `7 / 2` | **3** (целочисленное!) |
| `%` | остаток от деления | `7 % 2` | 1 |

> ⚠️ **Ловушка:** `7 / 2` для целых даёт `3`, а не `3.5`! Дробная часть отбрасывается.
> Чтобы получить `3.5`, хотя бы одно число должно быть дробным: `7.0 / 2`.

```c
int a = 7, b = 2;
printf("%d\n", a / b);          // 3
printf("%f\n", 7.0 / 2);        // 3.500000
printf("%d\n", a % b);          // 1 (остаток)
```

💡 Оператор `%` (остаток) — супер-полезный. Чётность: `n % 2 == 0`. Каждый 5-й: `i % 5 == 0`.

## 📖 Операторы сравнения (дают «истину» или «ложь»)

| Оператор | Значение |
|----------|----------|
| `==` | равно |
| `!=` | не равно |
| `>` `<` | больше / меньше |
| `>=` `<=` | больше или равно / меньше или равно |

> ⚠️ **Самая частая ошибка в C:** перепутать `=` (присваивание) и `==` (сравнение)!
> ```c
> if (x = 5)   // ⚠️ ПРИСВОИЛ 5 переменной x! Условие всегда истинно.
> if (x == 5)  // ✅ СРАВНИЛ x с 5
> ```
> Компилятор с `-Wall` тебя предупредит. Ещё трюк: пиши `if (5 == x)` — тогда опечатка
> `if (5 = x)` не скомпилируется.

В C нет отдельного типа «логический». **0 = ложь, любое другое число = истина.**

## 📖 Логические операторы

| Оператор | Значение | Пример |
|----------|----------|--------|
| `&&` | И (оба условия истинны) | `age > 18 && age < 65` |
| `\|\|` | ИЛИ (хотя бы одно) | `day == 6 \|\| day == 7` |
| `!` | НЕ (инверсия) | `!isReady` |

## 📖 Условие `if / else if / else`

```c
int score = 75;

if (score >= 90) {
    printf("Отлично\n");
} else if (score >= 70) {
    printf("Хорошо\n");
} else if (score >= 50) {
    printf("Удовлетворительно\n");
} else {
    printf("Неудовлетворительно\n");
}
```

🖼️ Как это работает — поток выполнения:

```
score >= 90 ? ──да──► "Отлично"
     │ нет
     ▼
score >= 70 ? ──да──► "Хорошо"
     │ нет
     ▼
score >= 50 ? ──да──► "Удовл."
     │ нет
     ▼
            "Неудовл."
```

> 💡 Условия проверяются **сверху вниз**, выполнится **первое подходящее**, остальные
> пропускаются.

## 📖 Тернарный оператор (короткий if)

```c
int max = (a > b) ? a : b;
//         условие  ? если истина : если ложь
```

Эквивалент:
```c
int max;
if (a > b) max = a;
else max = b;
```

## 📖 `switch` — выбор из многих вариантов

Когда сравниваешь одну переменную с кучей конкретных значений:

```c
int day = 3;

switch (day) {
    case 1: printf("Понедельник\n"); break;
    case 2: printf("Вторник\n");     break;
    case 3: printf("Среда\n");       break;
    case 6:
    case 7: printf("Выходной!\n");   break;   // 6 и 7 → одно действие
    default: printf("Неизвестный день\n");
}
```

> ⚠️ **Не забывай `break`!** Без него выполнение «провалится» в следующий `case`
> (fall-through). Иногда это используют специально (как для 6 и 7), но чаще это баг.

## 🧱 Блоки и фигурные скобки

```c
if (x > 0)
    printf("Положительное\n");   // без {} — только ОДНА строка относится к if

if (x > 0) {
    printf("Положительное\n");   // с {} — весь блок относится к if
    printf("и ещё строка\n");
}
```

💡 **Совет:** всегда ставь `{}`, даже для одной строки. Это спасает от классических багов,
когда добавляешь вторую строку и забываешь скобки.

## ✅ Задачи

1. **Чёт/нечет.** Считай число, скажи чётное оно или нечётное (используй `%`).
2. **Максимум из трёх.** Считай три числа, выведи наибольшее.
3. **Калькулятор-1.** Считай два числа и символ операции (`+ - * /`), выведи результат
   через `switch`. Обработай деление на ноль.
4. **Год високосный?** Считай год. Високосный, если делится на 4, но не на 100, либо
   делится на 400.
5. **Оценка по баллам.** Считай балл 0–100 и выведи буквенную оценку (A/B/C/D/F).
6. **Знак числа.** Используя тернарный оператор, выведи `-1`, `0` или `1` в зависимости
   от знака числа.
