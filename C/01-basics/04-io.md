# 04 · Ввод и вывод 🛠️

> 🎯 **Цель блока:** научиться выводить данные на экран и считывать ввод пользователя.
> Здесь же впервые встретим символ `&` — «адрес переменной» (намёк на указатели!).

---

## 📖 Вывод: `printf`

`printf` печатает текст. Внутри строки можно вставлять значения через **спецификаторы**:

```c
int age = 25;
double height = 1.75;
char grade = 'A';

printf("Возраст: %d лет\n", age);
printf("Рост: %.2f м\n", height);   // .2 = два знака после запятой
printf("Оценка: %c\n", grade);
printf("Сразу всё: %d, %.2f, %c\n", age, height, grade);
```

Вывод:
```
Возраст: 25 лет
Рост: 1.75 м
Оценка: A
Сразу всё: 25, 1.75, A
```

🖼️ Как `printf` подставляет значения:

```
printf("Возраст: %d лет\n", age);
                   └┬┘            │
                    └─────────────┘
              на место %d подставится значение age (25)
```

### Полезные управляющие символы

| Символ | Значение |
|--------|----------|
| `\n` | новая строка |
| `\t` | табуляция (отступ) |
| `\\` | сам символ `\` |
| `\"` | кавычка внутри строки |
| `%%` | сам символ `%` |

## 📖 Ввод: `scanf`

`scanf` считывает то, что пользователь набрал с клавиатуры, и **кладёт в переменную**.

```c
#include <stdio.h>

int main(void)
{
    int age;
    printf("Сколько тебе лет? ");
    scanf("%d", &age);              // ← обрати внимание на &
    printf("Тебе %d лет!\n", age);
    return 0;
}
```

---

## ⭐ Самое важное: зачем `&` в scanf

```c
scanf("%d", &age);
            └┬─┘
        АДРЕС переменной age
```

`&age` означает **«адрес ячейки памяти, где лежит age»**.

🖼️ Почему адрес, а не сама переменная? Потому что `scanf` должен **записать** значение
в переменную. Чтобы что-то записать в ячейку — нужно знать её **адрес** (номер домика):

```
Без &:  scanf получает копию значения  → записать некуда → ❌
С  &:   scanf получает адрес ячейки     → знает куда писать → ✅

     age
   ┌──────┐
   │  ??? │ ← scanf пишет сюда, потому что знает адрес &age
   └──────┘
    0x1000
```

> ⚠️ **Ошибка №1 новичка:** забыть `&` в `scanf`. Программа скомпилируется (с
> предупреждением!), но при запуске **упадёт** или испортит память. Всегда ставь `&`
> перед переменной в `scanf` (кроме строк — там отдельная история, см. блок про строки).

💡 Этот `&` — твой первый шаг к **указателям**. В блоке про память мы раскроем эту тему
полностью. Пока запомни: `&переменная` = «её адрес».

## 🧪 Считываем несколько значений

```c
int a, b;
printf("Введи два числа через пробел: ");
scanf("%d %d", &a, &b);
printf("Сумма = %d\n", a + b);
```

Считать число с дробью и символ:
```c
double price;
char currency;
scanf("%lf %c", &price, &currency);   // %lf — для double в scanf!
```

> ⚠️ Тонкость: в `printf` для `double` используется `%f`, а в `scanf` — `%lf`. Запомни это
> различие, иначе ввод дробных чисел не сработает.

---

## ⚠️ Проверяй результат scanf

`scanf` возвращает, **сколько значений** удалось прочитать. Профи это проверяют:

```c
int n;
if (scanf("%d", &n) != 1) {
    printf("Ошибка: это не число!\n");
    return 1;
}
```

Если пользователь введёт «abc» вместо числа — без проверки в `n` останется мусор.

1. **Анкета.** Спроси у пользователя имя (одно слово, `%s`), возраст и рост. Выведи
   красиво анкету.
2. **Сумма и среднее.** Считай три числа, выведи их сумму и среднее арифметическое
   (с двумя знаками после запятой).
3. **Конвертер.** Считай температуру в Цельсиях (`double`), выведи в Фаренгейтах:
   `F = C * 9/5 + 32`.
4. **Площадь.** Считай длину и ширину прямоугольника, выведи площадь и периметр.
5. **Время.** Считай количество секунд, выведи сколько это часов, минут и секунд.
   *(Подсказка: деление `/` и остаток `%`.)*
