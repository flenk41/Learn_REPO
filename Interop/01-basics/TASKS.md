# ✅ Задачи · Уровень 1 (FFI: основы)

> Тебе понадобится компилятор C (из [C-трека](../../C/00-setup/01-installation.md)) и Python.
> Сложность: ⭐ · ⭐⭐ · ⭐⭐⭐. Удобнее на Linux/WSL.

---

## 🟢 Блок A — первый FFI

1. ⭐ **multiply** — C-функция, собери в .so/.dll, вызови из Python (ctypes).
2. ⭐ **Без сигнатуры** — убери argtypes/restype, увидь искажение.
3. ⭐ **3 функции** — библиотека из нескольких математических функций.
4. ⭐⭐ **Замер** — тяжёлый цикл Python vs C через FFI.

---

## 🟢 Блок B — Python + C

5. ⭐⭐ **sum_array/max_array** — массив из Python в C.
6. ⭐⭐ **Изменение на месте** — C удваивает элементы массива.
7. ⭐⭐ **Структура** — опиши Point/Rectangle в Python, передай в C.
8. ⭐⭐⭐ **Cython** (по желанию) — ускорь функцию, сравни подходы.

---

## 🟢 Блок C — данные

9. ⭐⭐ **Строка туда-обратно** (encode/decode, кириллица).
10. ⭐⭐ **byref** — C меняет число по указателю.
11. ⭐⭐⭐ **Раскладка-детектив** — перепутай поля структуры, объясни мусор.

---

## 🟢 Блок D — владение

12. ⭐⭐ **Caller-allocated** буфер.
13. ⭐⭐ **make/free** — парные функции, корректное освобождение.
14. ⭐⭐⭐ **use-after-free через границу** — поймай (ASan).

---

## 🟢 Блок E — безопасность

15. ⭐⭐ **Код ошибки → исключение**.
16. ⭐⭐ **Проверка NULL**.
17. ⭐⭐⭐ **Безопасная обёртка-класс** над C-библиотекой.

---

## 🧩 Где тренироваться

- [Python ctypes docs](https://docs.python.org/3/library/ctypes.html) — официальная документация.
- [cffi](https://cffi.readthedocs.io) — альтернатива ctypes, удобнее для сложного.
- [Real Python: C extensions](https://realpython.com/python-bindings-overview/) — обзор подходов.

---

## 🏁 Критерий «зачёт»

- Блок A — все, B — минимум 2 (включая массив).
- Блок C — минимум 2, D — make/free + владение.
- Блок E — безопасная обёртка обязательно.
