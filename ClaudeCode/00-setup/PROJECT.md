# 🚀 Пет-проект уровня 0 — «Песочница агента»

> 🎯 **Цель:** пройти весь путь «установил → запустил → дал агенту первую задачу» и
> убедиться, что Claude Code работает у тебя на машине.

---

## 📋 Что делаем

Создаём маленький учебный репозиторий и проводим в нём первую сессию Claude Code, не боясь
что-то сломать.

### Шаг 1. Песочница

Создай папку и пару простых файлов (например, маленький скрипт + README):

```
sandbox/
├── README.md      (2-3 строки про проект)
└── hello.py       (выводит "Hello")  — или любой язык
```

### Шаг 2. Запуск

```bash
cd sandbox
claude
```

Выполни `/help`, `/model`, `/status` — осмотрись.

### Шаг 3. Первая задача агенту

Попроси агента (обычным текстом) что-нибудь безопасное, например:

- «Опиши, что находится в этой папке».
- «Добавь в `hello.py` функцию, которая здоровается по имени, и покажи изменения».

Посмотри, как он **читает файлы**, **предлагает правки** и **спрашивает разрешение** на
действия. Не принимай правки вслепую — прочитай, что он хочет сделать.

### Шаг 4. Рефлексия

Запиши в `README.md` песочницы 3 наблюдения: что приятно удивило, что насторожило, где
агент мог ошибиться.

---

## ✅ Критерии готовности

- [ ] Создал учебную песочницу с файлами
- [ ] Запустил `claude`, осмотрелся командами
- [ ] Дал агенту первую задачу и проследил его цикл (чтение → правка → подтверждение)
- [ ] Записал наблюдения

## 🌟 Усложнение (по желанию)

- Попробуй ту же задачу на двух моделях (`/model`) и сравни результат и скорость.
- Попроси агента **сначала объяснить план**, а потом делать — почувствуй разницу
  (это подготовка к Plan Mode из уровня 1).
