# 🚀 Финальные пет-проекты — стань C++ Senior

> 🎯 **Цель:** выбери **один-два** серьёзных проекта, объединяющих всё: классы, шаблоны,
> STL, умные указатели, move, многопоточность, современный C++. Это проекты для портфолио.

> 💡 Каждый — 1–4 недели. Веди в git, собирай через CMake, покрой тестами, используй
> санитайзеры, придерживайся правила нуля.

## 🏆 Проект 1 — Своя библиотека контейнеров (рекомендую)

Реализуй свои `Vector<T>`, `LinkedList<T>`, `HashMap<K,V>` — как в STL.

**Что внутри:**
- 🧩 **Шаблоны** (модуль 15) — контейнеры для любого типа.
- 🧠 **Управление памятью**: rule of five/zero, move-семантика (модули 19).
- 🔁 **Итераторы**, совместимые с range-for и алгоритмами STL.
- ✅ Тесты, ASan, бенчмарки против std-контейнеров.

**Прокачивает:** шаблоны, память, итераторы — глубочайшее понимание STL изнутри.

2D-игра (платформер, змейка, тетрис) с графикой.

## 🏆 Проект 3 — Интерпретатор / калькулятор выражений

Программа, разбирающая и вычисляющая выражения (`(3 + 4) * 2`).

**Что внутри:**
- 🔤 Лексер и парсер → AST (дерево на `unique_ptr`).
- 🌳 Полиморфные узлы (`virtual evaluate()`).
- 🧠 RAII/умные указатели для дерева — ноль ручного delete.
- ➕ Переменные, функции, возможно своя стековая виртуальная машина.

**Прокачивает:** деревья, полиморфизм, память — фундамент компиляторов.

## 🏆 Проект 4 — Многопоточный обработчик данных

Инструмент, параллельно обрабатывающий большой объём данных (лог-анализатор, обработчик
изображений).

**Что внутри:**

- 🔒 Синхронизация (mutex/atomic), TSan-чистота.
- ⚡ Профилирование, оптимизация, замер ускорения.

**Прокачивает:** параллелизм, производительность.

## 🏆 Проект 5 — Key-Value база данных в памяти

Мини-Redis: хранилище ключ→значение с сохранением на диск.

**Что внутри:**
- 🗃️ `unordered_map` + своя структура хранения.
- 💾 Сериализация в файл, загрузка.
- 🧵 Потокобезопасный доступ.
- 🌐 (по желанию) сетевой интерфейс (сокеты).

**Прокачивает:** структуры данных, память, архитектуру систем.

## 📋 Требования к ЛЮБОМУ финальному проекту

- [ ] **CMake**-сборка, стандарт C++17/20.
- [ ] ООП-архитектура, разбит на модули (`.h`/`.cpp`).
- [ ] **Умные указатели и RAII** — ноль голых `new`/`delete`, правило нуля.
- [ ] `-Wall -Wextra`, чистые санитайзеры (ASan/UBSan; TSan для многопоточных).
- [ ] **Тесты** (Catch2/GoogleTest).
- [ ] Обработка ошибок (исключения/optional), не падает на кривом вводе.
- [ ] README, git с осмысленными коммитами.
- [ ] Где уместно — замеры производительности.

## 🎓 Что дальше

Ты прошёл три языка — **C** (ручная память), **C++** (контроль + безопасность через RAII),
**Python** (автоматическая память). Теперь ты понимаешь память на всех уровнях и видишь,
как разные языки решают одну задачу по-разному.

Следующие направления:
- **Rust** — безопасность памяти без сборщика мусора и без RAII-ручного управления
  (borrow checker). После C++ ты особенно оценишь его идеи владения.
- **Углубление C++:** корутины, модули (C++20), метапрограммирование, конкретные домены
  (геймдев, HFT, embedded).
- **Специализация:** движки (Unreal), системное ПО, высоконагруженные системы.
