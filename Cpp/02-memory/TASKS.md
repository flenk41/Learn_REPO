## 🟢 Блок A — ссылки

1. ⭐ **Псевдоним** — измени переменную через ссылку.
2. ⭐ **swap по ссылке** (без указателей).
3. ⭐⭐ **Сумма вектора** по `const&` (без копий).
4. ⭐⭐ **Ссылка vs указатель** — одно изменение двумя способами, сравни синтаксис.

---

## 🟢 Блок B — new/delete и опасности

5. ⭐ **new/delete** для int и массива.
6. ⭐⭐ **Поймай утечку** (ASan).
7. ⭐⭐ **Use-after-free и double-free** — воспроизведи и поймай.
8. ⭐⭐ **Исключение ломает delete** — покажи утечку, потом почини через RAII/vector.

---

## 🟢 Блок C — RAII

9. ⭐ **Класс с деструктором** — проследи автоматический вызов.
10. ⭐⭐ **Порядок разрушения** (LIFO) для нескольких объектов.
11. ⭐⭐ **RAII-массив** (`IntArray`) — без утечек под ASan.
12. ⭐⭐ **RAII при исключении** — деструктор всё равно вызывается.
13. ⭐⭐⭐ **RAII-файл** — обёртка над `fopen`/`fclose`.

---

## 🟢 Блок D — умные указатели

14. ⭐ **unique_ptr** через `make_unique`.
15. ⭐⭐ **Передача владения** через `std::move`.
16. ⭐⭐ **shared_ptr + use_count()** в блоке.
17. ⭐⭐⭐ **Циклическая утечка** shared_ptr → почини `weak_ptr`.
18. ⭐⭐⭐ **Связный список на unique_ptr** (без ручного delete).

---

## 🟢 Блок E — move

19. ⭐⭐ **Move vs копия** — замер времени на большом векторе.
20. ⭐⭐ **push_back с move** — исходные строки пустеют.
