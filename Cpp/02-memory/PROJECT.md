# 🚀 Пет-проект уровня 2 — Свой умный указатель (MyUniquePtr)

> 🎯 **Цель:** реализовать упрощённый аналог `std::unique_ptr` — RAII-обёртку над сырым
> указателем. Сделав это, ты до конца поймёшь RAII, move-семантику и владение — самую суть
> памяти C++.

> 💪 Это серьёзный проект, объединяющий весь Уровень 2. После него умные указатели для
> тебя — открытая книга.

---

## 📖 Что делаем

Класс-шаблон `MyUniquePtr<T>`, который:
- владеет объектом в куче (захват в конструкторе);
- освобождает его в деструкторе (RAII);
- **нельзя копировать** (единоличное владение);
- **можно перемещать** (передача владения через move);
- ведёт себя как указатель (`*`, `->`).

```cpp
MyUniquePtr<int> p(new int(42));
std::cout << *p << "\n";          // 42
MyUniquePtr<int> p2 = std::move(p);  // передать владение, p станет пустым
// delete не нужен — деструктор сам освободит
```

---

## ✅ Требования (минимум)

- [ ] Шаблон `template<typename T> class MyUniquePtr`.
- [ ] Конструктор принимает `T*` и владеет им.
- [ ] Деструктор делает `delete`.
- [ ] Операторы `*` (разыменование) и `->` (доступ к членам).
- [ ] **Запрет копирования** (`= delete` для копирующего конструктора и присваивания).
- [ ] **Перемещающий** конструктор и присваивание (крадут указатель, обнуляют источник).
- [ ] Метод `get()` (вернуть сырой указатель) и `release()`/`reset()` по желанию.
- [ ] **Ноль утечек** под `-fsanitize=address`.

---

## 🪜 Пошаговый план

1. **Каркас.** Шаблон с приватным полем `T* ptr;`. Конструктор `MyUniquePtr(T* p)`.
2. **RAII.** Деструктор `~MyUniquePtr() { delete ptr; }`.
3. **Доступ.** `T& operator*()` и `T* operator->()`.
4. **Запрет копий.** Удали копирующий конструктор и оператор присваивания (`= delete`).
5. **Move.** Перемещающий конструктор: забрать `ptr` у источника, обнулить его.
   Перемещающее присваивание: освободить своё, забрать чужое, обнулить источник.
6. **Тесты.** Создай, перемести, проверь, что источник пуст; убедись, что копирование не
   компилируется.
7. **ASan.** Чистый прогон.

## 🧱 Скелет

```cpp
#include <iostream>
#include <utility>   // std::exchange

template<typename T>
class MyUniquePtr {
    T* ptr;
public:
    // конструктор — захват владения
    explicit MyUniquePtr(T* p = nullptr) : ptr(p) {}

    // деструктор — освобождение (RAII)
    ~MyUniquePtr() { delete ptr; }

    // ЗАПРЕТ копирования (единоличное владение)
    MyUniquePtr(const MyUniquePtr&) = delete;
    MyUniquePtr& operator=(const MyUniquePtr&) = delete;

    // перемещающий конструктор — забрать указатель, обнулить источник
    MyUniquePtr(MyUniquePtr&& other) noexcept
        : ptr(std::exchange(other.ptr, nullptr)) {}

    // перемещающее присваивание
    MyUniquePtr& operator=(MyUniquePtr&& other) noexcept {
        if (this != &other) {
            delete ptr;                              // освободить своё
            ptr = std::exchange(other.ptr, nullptr); // забрать чужое
        }
        return *this;
    }

    // поведение указателя
    T& operator*()  const { return *ptr; }
    T* operator->() const { return ptr; }
    T* get()        const { return ptr; }
    explicit operator bool() const { return ptr != nullptr; }
};

int main() {
    MyUniquePtr<int> p(new int(42));
    std::cout << *p << "\n";              // 42

    MyUniquePtr<int> p2 = std::move(p);  // передача владения
    std::cout << (p ? "p владеет" : "p пуст") << "\n";   // p пуст
    std::cout << *p2 << "\n";            // 42
    // деструкторы сами всё освободят
    return 0;
}
```
```bash
g++ -std=c++17 -Wall -Wextra -fsanitize=address -g myptr.cpp -o myptr
```

## 💪 Усложнения (для «отлично»)

- ➕ `reset(T* p = nullptr)` — заменить управляемый объект.
- ➕ `release()` — отдать владение наружу (вернуть указатель, обнулить свой).
- ➕ Поддержка массивов (`delete[]`) через отдельную специализацию или удалитель.
- ➕ Кастомный удалитель (deleter) — функция, вызываемая вместо `delete`.
- ➕ Простой `MySharedPtr` со счётчиком ссылок (сложнее — для амбициозных).
- ➕ Юнит-тесты: проверь владение, перемещение, что копирование не компилируется.
