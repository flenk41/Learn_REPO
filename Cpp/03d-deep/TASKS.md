# ✅ Задачи раздела · Углублённый C++

> Продвинутые темы. Используй godbolt, чтобы видеть, во что компилируются шаблоны.

---

## 🟢 Базовые

1. **Fold.** Через fold expressions: `sum`, `product`, `all_true` для любого числа аргументов.
2. **Концепт.** Определи `Numeric` и `Printable`, ограничь ими функции. Проверь ясность ошибок.
3. **Type traits.** Используй `is_integral_v`, `is_same_v`, `remove_const_t` в примерах.

## 🟡 Средние

4. **make_unique.** Свой `make_unique` с perfect forwarding пакета.
5. **if constexpr.** `to_string`, выбирающий реализацию по типу (число/строка/контейнер).
6. **Перегрузка по концептам.** `process` с версиями для `std::integral` и `std::floating_point`.
7. **STL-бенчмарк.** `vector` vs `list` (вставка/обход), `map` vs `unordered_map` (поиск). Объясни устройством.

## 🔴 Со звёздочкой

8. ⭐ **Свой trait + эволюция.** `is_pointer` через специализацию; перепиши SFINAE-пример через
   `if constexpr`, потом через концепт.
9. ⭐ **Ranges-конвейер.** filter→transform→take через `|`; проверь ленивость и отсутствие копий.
10. ⭐ **Инвалидация.** Воспроизведи и объясни баг с указателем в `vector` после реаллокации.
11. ⭐⭐ **Свой контейнер с концептами.** Мини-контейнер, шаблонный, ограниченный концептами, с
    итераторами (работает в range-for и с `<algorithm>`).
