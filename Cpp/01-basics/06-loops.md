# 06 · Циклы и range-based for

> 🎯 **Цель блока:** освоить циклы C++ — классические как в C, плюс удобный современный
> range-based for.

---

## 🔁 Классические циклы (как в C)

```cpp
// for
for (int i = 0; i < 5; i++) {
    std::cout << i << " ";          // 0 1 2 3 4
}

// while
int n = 5;
while (n > 0) {
    std::cout << n << " ";
    n--;
}

// do-while
int x;
do {
    std::cout << "Введи положительное: ";
    std::cin >> x;
} while (x <= 0);
```

`break` (выйти) и `continue` (пропустить итерацию) работают как в C.

---

## ⭐ range-based for — главная новинка

Перебор коллекции **без индексов** (как `for ... in` в Python):

```cpp
#include <vector>

std::vector<int> nums = {10, 20, 30, 40};

for (int n : nums) {                // для каждого n из nums
    std::cout << n << " ";          // 10 20 30 40
}
```

🖼️
```
   for (int n : nums)
            │     └─ коллекция
            └─ n по очереди = каждый элемент
```

### С `auto` и ссылкой
```cpp
// читаем (копия каждого элемента)
for (auto n : nums) { std::cout << n; }

// читаем без копирования (const-ссылка) — для больших объектов эффективнее
for (const auto& n : nums) { std::cout << n; }

// ИЗМЕНЯЕМ элементы (ссылка &)
for (auto& n : nums) { n *= 2; }    // удвоить каждый элемент на месте
```

> 💡 **Правило:** хочешь только читать — `const auto&` (без копий). Хочешь менять —
> `auto&`. Это важно для производительности (Уровень 4) и встречается постоянно.

## 🔁 Перебор строки

```cpp
std::string s = "Привет";
for (char c : s) {
    std::cout << c << " ";
}
```

---

## 📖 Вложенные циклы

```cpp
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        std::cout << i * j << "\t";
    }
    std::cout << "\n";
}
```
