# 🚀 Пет-проект уровня 1 — Консольный калькулятор

> 🎯 **Цель:** собрать всё из уровня 1 (потоки, условия, циклы, функции, перегрузка) в
> одну законченную программу на C++.

---

## 📋 Что должна уметь программа

Калькулятор в цикле с меню, пока пользователь не выйдет:

```
==== КАЛЬКУЛЯТОР ====
1. Сложение
2. Вычитание
3. Умножение
4. Деление
0. Выход
Выбор: 1
Два числа: 10 5
10 + 5 = 15
```

---

## ✅ Требования (минимум)

- [ ] Меню в цикле (`while`/`do-while`), выход по `0`.
- [ ] Операции `+ − × ÷` (+ остаток по желанию), каждая — **отдельная функция**.
- [ ] Выбор через `switch`.
- [ ] Обработка деления на ноль.
- [ ] Корректный ввод через `std::cin` с проверкой.
- [ ] Компилируется без предупреждений (`-Wall -Wextra`).

---

## 🪜 Пошаговый план

1. **Каркас.** `main` с циклом, печать меню, чтение выбора.
2. **Выход** по `0` → `break`.
3. **Функции операций:** `double add(double, double)` и т.д.
4. **switch** связывает пункты меню с функциями.
5. **Защита:** деление на ноль, проверка `if (std::cin >> x)`.
6. **Полировка** вывода.

---

## 🧱 Скелет

```cpp
#include <iostream>

double add(double a, double b) { return a + b; }
double sub(double a, double b) { return a - b; }
double mul(double a, double b) { return a * b; }
// divide реализуй сам, обработай b == 0

void printMenu() {
    std::cout << "\n==== КАЛЬКУЛЯТОР ====\n"
              << "1. Сложение\n2. Вычитание\n3. Умножение\n"
              << "4. Деление\n0. Выход\nВыбор: ";
}

int main() {
    int choice;
    while (true) {
        printMenu();
        if (!(std::cin >> choice)) { std::cout << "Введи число!\n"; break; }
        if (choice == 0) { std::cout << "Пока!\n"; break; }

        double a, b;
        std::cout << "Два числа: ";
        std::cin >> a >> b;

        switch (choice) {
            case 1: std::cout << "= " << add(a, b) << "\n"; break;
            case 2: std::cout << "= " << sub(a, b) << "\n"; break;
            // дополни
            default: std::cout << "Нет такой операции\n";
        }
    }
    return 0;
}
```

## 💪 Усложнения (для «отлично»)

- ➕ Степень и корень (`#include <cmath>`).
- ➕ Память калькулятора (M+, MR, MC).
- ➕ История последних операций (используй `std::vector`).
- ➕ Перегрузка функций для целых и дробных.
- ➕ Счётчик операций при выходе.
