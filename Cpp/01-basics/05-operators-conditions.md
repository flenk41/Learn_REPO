# 05 · Операторы и условия

> 🎯 **Цель блока:** научиться вычислять выражения и принимать решения. Почти всё как в C,
> с парой приятных дополнений C++.

---

## 📖 Арифметика

| Оператор | Действие | `7 ? 2` |
|----------|----------|---------|
| `+ - *` | плюс, минус, умножение | |
| `/` | деление | `7 / 2` = **3** (для целых!) |
| `%` | остаток | `7 % 2` = 1 |

> ⚠️ Как и в C: `7 / 2` для целых = `3`. Для `3.5` нужна дробь: `7.0 / 2`.

```cpp
std::cout << 7 / 2 << "\n";      // 3
std::cout << 7.0 / 2 << "\n";    // 3.5
std::cout << 7 % 2 << "\n";      // 1
```

## 📖 Сравнение и логика

| Сравнение | Логика |
|-----------|--------|
| `==` `!=` | `&&` (И) |
| `>` `<` | `\|\|` (ИЛИ) |
| `>=` `<=` | `!` (НЕ) |

```cpp
bool adult = age >= 18 && age < 65;
bool weekend = day == 6 || day == 7;
```

🆚 В C++ результат сравнения — настоящий `bool` (`true`/`false`), а не `int` как в C.

> ⚠️ Та же классическая ловушка: не путай `=` (присваивание) и `==` (сравнение)!
> Компилятор с `-Wall` предупредит.

## 📖 Условие `if / else if / else`

```cpp
int score = 75;

if (score >= 90) {
    std::cout << "Отлично\n";
} else if (score >= 70) {
    std::cout << "Хорошо\n";
} else {
    std::cout << "Нужно подтянуть\n";
}
```

### ⭐ if с инициализацией (C++17)
Можно объявить переменную прямо в `if` — она видна только внутри:
```cpp
if (int n = get_value(); n > 0) {
    std::cout << n << " положительное\n";
}   // n существует только здесь
```
💡 Удобно: переменная не «утекает» в окружающий код.

## 📖 Тернарный оператор

```cpp
int max = (a > b) ? a : b;       // если a>b, то a, иначе b
std::string s = (age >= 18) ? "взрослый" : "ребёнок";
```

## 📖 `switch`

```cpp
switch (day) {
    case 1: std::cout << "Понедельник\n"; break;
    case 6:
    case 7: std::cout << "Выходной\n"; break;   // 6 и 7 → одно действие
    default: std::cout << "Будний день\n";
}
```

> ⚠️ Не забывай `break` — иначе «провал» в следующий `case` (как в C).
