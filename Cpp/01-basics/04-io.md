# 04 · Потоки ввода-вывода

> 🎯 **Цель блока:** научиться выводить и считывать данные через потоки `cout`/`cin` —
> «питоничный» по удобству, но типобезопасный способ C++.

---

## 📖 Вывод: `std::cout`

```cpp
#include <iostream>

int main() {
    int age = 25;
    double height = 1.75;
    std::string name = "Гена";

    std::cout << "Имя: " << name << "\n";
    std::cout << "Возраст: " << age << ", рост: " << height << "\n";
    std::cout << "Сумма: " << (2 + 3) << std::endl;
    return 0;
}
```

🖼️ Оператор `<<` «отправляет» значения в поток слева направо:
```
   std::cout << "Возраст: " << age << "\n";
              └── отправляем по очереди: текст, потом число, потом перенос
```

💡 Не нужно помнить `%d`/`%f`/`%s` как в C — поток сам понимает тип каждого значения.

### `\n` vs `std::endl`
- `"\n"` — просто перевод строки (быстрее).
- `std::endl` — перевод строки **+ сброс буфера** (медленнее, но гарантирует вывод).

> 💡 Привыкай к `"\n"` в обычных случаях, `std::endl` — когда нужно немедленно увидеть
> вывод (например перед возможным крашем).

## 📖 Ввод: `std::cin`

```cpp
#include <iostream>

int main() {
    int age;
    std::cout << "Сколько тебе лет? ";
    std::cin >> age;                    // оператор >> читает из потока В переменную
    std::cout << "Тебе " << age << " лет\n";
    return 0;
}
```

🖼️ Оператор `>>` «извлекает» из потока в переменную:
```
   std::cin >> age;
            └── прочитать значение и положить в age
```

🆚 В C тебе нужен был `&` (`scanf("%d", &age)`). В C++ `cin >> age` — **без `&`**, проще
и безопаснее. (Под капотом работают ссылки — узнаешь в Уровне 2.)

### Считать несколько значений
```cpp
int a, b;
std::cout << "Два числа: ";
std::cin >> a >> b;                     // можно цепочкой
std::cout << "Сумма: " << a + b << "\n";
```

## ⭐ Считывание строк: `getline`

`std::cin >> name` читает только **одно слово** (до пробела). Чтобы прочитать строку
целиком (с пробелами) — `std::getline`:

```cpp
#include <iostream>
#include <string>

int main() {
    std::string full_name;
    std::cout << "Полное имя: ";
    std::getline(std::cin, full_name);   // читает всю строку, включая пробелы
    std::cout << "Привет, " << full_name << "\n";
    return 0;
}
```

> ⚠️ Тонкость: если смешиваешь `cin >> x` и `getline`, после `>>` в буфере остаётся
> символ перевода строки, и `getline` прочитает пустую строку. Лечится `std::cin.ignore()`
> между ними. Пока запомни: для целых строк используй `getline`.

## ⚠️ Проверка ввода

`cin` возвращает значение, которое в условии становится `false` при ошибке ввода:

```cpp
int n;
if (std::cin >> n) {
    std::cout << "Ок: " << n << "\n";
} else {
    std::cout << "Это не число!\n";
}
```

## 📖 Форматирование вывода

```cpp
#include <iostream>
#include <iomanip>          // для манипуляторов

int main() {
    double pi = 3.14159265;
    std::cout << std::fixed << std::setprecision(2) << pi << "\n";  // 3.14
    std::cout << std::setw(8) << 42 << "\n";   // выравнивание по ширине 8
    return 0;
}
```
