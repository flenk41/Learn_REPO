# ✅ Задачи · Проекты и API (C++)

> Собирай через CMake, компилируй с `-Wall -Wextra -fsanitize=address`. Сложность: ⭐ · ⭐⭐ · ⭐⭐⭐

---

## 🟢 Блок A — структура проекта

1. ⭐ **Раздели монолит** на `main.cpp` + класс в `.hpp`/`.cpp`.
2. ⭐⭐ **CMake** — раскладка `include/ src/` + `CMakeLists.txt`, сборка через `cmake -B build`.
3. ⭐⭐ **Библиотека** — оформи логику как `add_library`, подключи к программе.
4. ⭐⭐ **namespace** — оберни код, спрячь внутреннее в анонимный namespace.
5. ⭐⭐⭐ **Три модуля** с правильным направлением зависимостей + CMake + тесты.

---

## 🟢 Блок B — проектирование API

6. ⭐⭐ **Инкапсуляция** — класс с public-API и private-состоянием.
7. ⭐⭐ **Владение типами** — функции с `unique_ptr`/`const&`/`optional`.
8. ⭐⭐ **const correctness** — расставь `const` на всех неизменяющих методах.
9. ⭐⭐⭐ **pImpl** — спрячь реализацию класса за `unique_ptr<Impl>`.
10. ⭐⭐⭐ **API-ревью** — понятен ли заголовок без `.cpp`.

---

## 🟢 Блок C — веб-API

11. ⭐⭐ **GET через cpr** к публичному API, статус + тело.
12. ⭐⭐ **Парсинг nlohmann/json** — достать поля.
13. ⭐⭐ **Обработка** 404, ошибок сети и битого JSON.
14. ⭐⭐ **Погода** через open-meteo (без ключа).
15. ⭐⭐⭐ **Свой клиент** — класс-обёртка над API с чистым интерфейсом и структурами-результатами.
