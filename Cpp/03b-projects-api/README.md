## 🎯 Что такое API — одним предложением

> **API — это «контракт»: набор классов, функций и правил, через которые с твоим кодом
> (или чужим сервисом) общаются, не зная, что у него внутри.**

Две стороны, разберём обе:

| Сторона | Пример | Где |
|---------|--------|-----|
| **Ты проектируешь API** | другие зовут твой `client.getUser()`, не зная про детали | модули 1–2 |
| **Ты используешь чужой API** | HTTP-запрос к сервису, получаешь JSON | модуль 3 |

💡 В обоих случаях суть одна — **граница между «что снаружи» и «как внутри»**. В C++ эту
границу проводят заголовки (`.hpp`), `public/private` и идиома pImpl.
