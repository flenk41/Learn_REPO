# 01 · Установка компилятора и редактора 🛠️

> 🎯 **Цель блока:** установить компилятор C++ и проверить его работу. В конце ты
> увидишь версию компилятора — значит всё готово.

---

## 📖 Что нам нужно

| Что | Зачем |
|-----|-------|
| **Компилятор C++** | превращает твой код в программу (`g++`, `clang++` или MSVC) |
| **Редактор VS Code** | удобно писать код |

Возьмём **g++** (часть GCC) — самый популярный компилятор C++.

---

## 🪟 Установка на Windows (твоя система)

### Вариант А — MSYS2 (рекомендую, как в C-курсе)

Если ты уже ставил GCC для курса C — у тебя **уже есть g++**! Проверь:
```powershell
g++ --version
```
Если показывает версию — переходи сразу к установке VS Code ниже.

Если нет — установи через MSYS2:

1. Скачай установщик с **https://www.msys2.org**, установи в `C:\msys64`.
2. В открывшемся терминале MSYS2 введи:
   ```bash
   pacman -S mingw-w64-ucrt-x86_64-gcc
   ```
   (пакет gcc включает и `g++` для C++).
3. Добавь в **PATH** путь `C:\msys64\ucrt64\bin` (Win → «переменные среды» →
   Path → Создать → вставить путь → ОК).

### Вариант Б — отдельный MinGW
Скачай готовую сборку MinGW-w64 (например с WinLibs: **https://winlibs.com**), распакуй,
добавь папку `bin` в PATH.

### Проверка ✅
Открой **новый** PowerShell и введи:
```powershell
g++ --version
```
Должно появиться что-то вроде:
```
g++ (Rev3, Built by MSYS2 project) 14.2.0
```
🎉 Компилятор C++ установлен!

> ⚠️ Если `g++ не найден` — путь в PATH прописан неверно или ты не открыл **новое** окно
> терминала. Проверь, что в `C:\msys64\ucrt64\bin` есть файл `g++.exe`.

## 🍎 macOS

```bash
xcode-select --install     # установит clang++ (совместим с g++)
clang++ --version
```
В курсе везде можешь писать `clang++` вместо `g++` — команды одинаковые.

---

## 🐧 Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install g++ gdb
g++ --version
```

---

## 💻 Установка редактора VS Code

1. Скачай с **https://code.visualstudio.com**, установи.
2. Открой Extensions (слева) → установи **C/C++** (от Microsoft) — подсветка,
   автодополнение, отладка. Это то же расширение, что для C.

> 💡 Расширение C/C++ работает и с C, и с C++ — ничего дополнительно ставить не нужно.

---

## 🗂️ Рабочая папка

Создай папку для кода курса, например:
```
D:\LearnProject\my-code\Cpp\
```
Открой её в VS Code (**Файл → Открыть папку**).

---

## 📖 Важно: стандарт языка

C++ развивается: C++11, 14, 17, 20, 23. Мы будем использовать **современный C++** —
компилируй с флагом стандарта:
```powershell
g++ -std=c++17 program.cpp -o program
```
`-std=c++17` (или `c++20`) включает современные возможности. Без него компилятор может
использовать старый стандарт, где нет умных указателей и прочего.
