# 🚀 Пет-проект уровня 0 — «Подготовка рабочего места трейдера»

> 🎯 **Цель:** настроить всё для безопасного обучения — демо-счёт, платформу, понимание
> инструмента и его рисков — до того, как смотреть на любые стратегии.

> ⚠️ Только демо. Никаких реальных денег на этом этапе.

---

## 📋 Что делаем

Готовим «рабочее место» и базовое понимание инструмента, чтобы дальше учиться осознанно.

### Шаги

1. **Демо-счёт.** Заведи демо у регулируемого брокера (или используй бесплатный график-сервис
   вроде TradingView для анализа). Проверь, что видишь живой график фьючерса.
2. **Платформа.** Освойся: график и таймфреймы, окно ордера (market/limit/stop), список позиций,
   индикаторы, инструменты рисования (линии, уровни).
3. **Паспорт инструмента.** Выбери один ликвидный фьючерс и выпиши его «паспорт»: базовый актив,
   размер контракта, шаг цены (тик), **стоимость тика**, дата экспирации, комиссия, спред.
4. **Риск-таблица.** Посчитай для этого инструмента: сколько денег = 10 тиков, 50 тиков, 100
   тиков движения. Это понадобится для стопов.
5. **Личные правила.** Напиши свои стартовые правила безопасности, например:
   - учусь только на демо, пока нет стабильных результатов;
   - на реал перейду с минимальных сумм;
   - никогда не рискую деньгами, которые не готов потерять;
   - не верю «сигналам» и «гуру» с обещаниями прибыли.

### Отчёт

`README.md` проекта: скриншот настроенной платформы, «паспорт» инструмента, риск-таблица, личные
правила. Ответь честно: сколько денег и времени ты готов потратить на обучение?

## ✅ Критерии готовности

- [ ] Настроен демо-счёт / график-сервис
- [ ] Освоена платформа (график, ордера, индикаторы)
- [ ] Составлен «паспорт» фьючерса со стоимостью тика
- [ ] Посчитана риск-таблица (тики → деньги)
- [ ] Записаны личные правила безопасности

## 🌟 Усложнение (по желанию)

- Сравни «паспорта» двух разных фьючерсов (например, индекс и сырьё) — где тик дороже, где
  волатильность выше? Какой инструмент рискованнее для новичка?
- Найди исторический пример резкого движения этого инструмента (гэп, обвал) и посчитай, что было
  бы с депозитом 1000$ на плече 1:10 без стопа. Это отрезвляет.
