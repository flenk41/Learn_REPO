## 🏆 Проект 1 — Асинхронный веб-скрейпер (рекомендую)

Инструмент, который параллельно скачивает и анализирует много страниц.

**Что внутри:**
- 🌐 **asyncio + aiohttp** — тысячи запросов в одном потоке (модуль 23).
- 🧠 Контроль памяти при больших объёмах (генераторы, модуль 20).
- 📊 Парсинг данных, сохранение в JSON/CSV.
- ⏱️ Профилирование, сравнение sync vs async по скорости (модуль 22).

**Прокачивает:** asyncio, работу с сетью, память, производительность.

## 🏆 Проект 2 — CLI-приложение с базой данных

Полноценная консольная программа (трекер финансов / задач / привычек) с хранением в SQLite.

**Что внутри:**
- 🗃️ **sqlite3** (встроен в Python) — реальная БД.
- 🏛️ ООП-архитектура: модели, репозитории, сервисы.
- 🎯 Аргументы командной строки (`argparse`).
- ✅ Тесты (pytest), типы (mypy), логирование.

**Прокачивает:** архитектуру, БД, профессиональную структуру проекта.

**Что внутри:**
- 🔢 **numpy / pandas** — векторизованная обработка (модуль 24).
- 🧠 Контроль памяти на больших данных (dtype, чанки, генераторы).
- 📈 Агрегации, статистика, экспорт результатов.
- ⏱️ Профилирование: оптимизация узких мест с доказательством.

**Прокачивает:** производительность, память, работу с данными.
