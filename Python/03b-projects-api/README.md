## 🎯 Что такое API — одним предложением

> **API — это «контракт»: набор функций/классов и правил, через которые с твоим кодом
> (или чужим сервисом) общаются, не зная, как он устроен внутри.**

Две стороны, разберём обе:

| Сторона | Пример | Где |
|---------|--------|-----|
| **Ты проектируешь API** | другие зовут твой `client.get_user()`, не зная про детали | модули 1–2 |
| **Ты используешь чужой API** | `requests.get(...)` к сервису, получаешь JSON | модуль 3 |

💡 В обоих случаях суть одна — **граница между «что снаружи» и «как внутри»**.
