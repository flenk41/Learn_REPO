## ⭐ `with` — правильный способ работы с файлами

```python
with open("data.txt", "w", encoding="utf-8") as f:
    f.write("Привет, файл!\n")
    f.write("Вторая строка\n")
# файл АВТОМАТИЧЕСКИ закрыт здесь, даже если была ошибка
```

🖼️ Что делает `with`:

```
   with open(...) as f:   ──► открыть файл (захватить ресурс)
       ...работа...
   (выход из блока)       ──► ГАРАНТИРОВАННО закрыть файл
                              даже если внутри случилась ошибка
```

> 💡 Это похоже на пару `malloc`/`free` из C, но автоматическую. `with` — это
> «контекстный менеджер»: он сам освобождает ресурс на выходе из блока. **Всегда**
> открывай файлы через `with`, чтобы не забыть закрыть и не потерять данные.

> ⚠️ Указывай `encoding="utf-8"` — иначе на Windows кириллица может ломаться.

---

## 📖 Режимы открытия

| Режим | Значение |
|-------|----------|
| `"r"` | чтение (по умолчанию) |
| `"w"` | запись (**стирает** существующий файл!) |
| `"a"` | дозапись в конец |
| `"r+"` | чтение и запись |
| `"rb"` / `"wb"` | бинарный режим |

> ⚠️ `"w"` удаляет содержимое файла! Для добавления используй `"a"`.

## 📖 Чтение файла

```python
# Весь файл одной строкой
with open("data.txt", encoding="utf-8") as f:
    content = f.read()

# Список строк
with open("data.txt", encoding="utf-8") as f:
    lines = f.readlines()

# ⭐ Построчно (лениво — файл это генератор строк, память не растёт!)
with open("data.txt", encoding="utf-8") as f:
    for line in f:
        print(line.strip())     # strip убирает \n в конце
```

💡 Перебор файла `for line in f` — ленивый (вспомни генераторы!). Так читают огромные
файлы, не загружая их целиком в память.

## 📖 Запись файла

```python
# Записать строки
with open("out.txt", "w", encoding="utf-8") as f:
    f.write("строка 1\n")
    f.writelines(["строка 2\n", "строка 3\n"])

# Дозаписать
with open("log.txt", "a", encoding="utf-8") as f:
    f.write("новое событие\n")

# print тоже умеет в файл
with open("out.txt", "w", encoding="utf-8") as f:
    print("через print", file=f)
```

## ⭐ Работа с JSON — самый частый формат

```python
import json

data = {"name": "Гена", "age": 30, "hobbies": ["читать", "код"]}

# Записать объект Python в файл как JSON
with open("user.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# Прочитать JSON обратно в объект Python
with open("user.json", encoding="utf-8") as f:
    loaded = json.load(f)
print(loaded["name"])           # Гена
```

🖼️
```
   объект Python  ──json.dump──►  файл .json (текст)
   {dict, list}   ◄──json.load──  (обратно в dict/list)
```

💡 JSON — стандарт обмена данными. `ensure_ascii=False` сохраняет кириллицу читаемой,
`indent=2` делает файл красивым.

## 📖 Работа с путями: pathlib

Современный способ работать с путями к файлам:

```python
from pathlib import Path

p = Path("data") / "users.txt"      # склейка путей (кроссплатформенно)
print(p.exists())                   # существует ли
print(p.suffix)                     # ".txt"
print(p.stem)                       # "users"

# Удобное чтение/запись
text = Path("data.txt").read_text(encoding="utf-8")
Path("out.txt").write_text("привет", encoding="utf-8")

# Перебор файлов в папке
for file in Path(".").glob("*.py"):
    print(file.name)
```

## 📖 Свой контекстный менеджер

`with` работает не только с файлами. Можно сделать свой:

```python
from contextlib import contextmanager
import time

@contextmanager
def timer(name):
    start = time.time()
    yield                            # здесь выполняется тело with
    print(f"{name}: {time.time() - start:.3f} сек")

with timer("Вычисление"):
    total = sum(range(10_000_000))
# автоматически напечатает время
```
