# 01 · Установка Python и редактора 🛠️

> 🎯 **Цель блока:** установить Python, проверить его работу и настроить редактор.
> В конце ты выполнишь команду и увидишь версию Python — значит всё готово.

---

## 📖 Что нам нужно

| Что | Зачем |
|-----|-------|
| **Python** | сам интерпретатор (он же выполняет твой код) |
| **pip** | менеджер пакетов (ставит чужие библиотеки) — идёт вместе с Python |
| **Редактор VS Code** | удобно писать код с подсветкой |

---

## 🪟 Установка на Windows (твоя система)

### Шаг 1. Скачать Python
1. Зайди на **https://www.python.org/downloads/**
2. Нажми большую кнопку **«Download Python 3.x»** (последняя версия).

### Шаг 2. Установить — ⚠️ ВАЖНЫЙ момент
Запусти скачанный файл. На **первом** экране установщика:

> ⚠️ **ОБЯЗАТЕЛЬНО поставь галочку внизу: «Add python.exe to PATH»!**
> Без неё команда `python` не будет работать в терминале, и ты потратишь час на
> непонятные ошибки. Это ошибка №1 новичков на Windows.

🖼️
```
┌─────────────────────────────────────────────┐
│   Install Python 3.x                         │
│                                              │
│   [ Install Now ]                            │
│   [ Customize installation ]                 │
│                                              │
│   ☑ Add python.exe to PATH   ← ⚠️ ВКЛЮЧИ!    │
└─────────────────────────────────────────────┘
```

Затем нажми **Install Now**, дождись окончания.

### Шаг 3. Проверить установку ✅
Открой **новый** PowerShell (нажми **Win**, набери `powershell`, Enter) и введи:

```powershell
python --version
```

Должно появиться что-то вроде:
```
Python 3.12.4
```

Проверь и pip:
```powershell
pip --version
```

🎉 Если обе команды показали версии — **Python установлен!**

> ⚠️ Если пишет `python не найден` или открывается Microsoft Store:
> - Ты не поставил галочку «Add to PATH» → переустанови, включив её.
> - Или попробуй команду `py --version` вместо `python` (на Windows работает спецзапускалка `py`).
> - Закрой и открой PowerShell **заново** после установки.

## 🍎 macOS

```bash
# Через Homebrew (если установлен)
brew install python

# Проверка

pip3 --version
```
На macOS команда часто `python3`, а не `python`.

## 🐧 Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv
python3 --version
```

## 💻 Установка редактора VS Code

1. Скачай с **https://code.visualstudio.com**, установи.
2. Открой VS Code → слева иконка **Extensions** (квадратики) → установи:
   - **Python** (от Microsoft) — подсветка, автодополнение, запуск, отладка.
   - *(автоматически подтянется Pylance — умные подсказки.)*

🖼️ После установки расширения в правом верхнем углу появится кнопка ▶️ (Run) — она
запускает текущий файл.

### Выбор интерпретатора в VS Code
Если VS Code не видит Python: нажми **Ctrl+Shift+P** → набери `Python: Select Interpreter`
→ выбери установленную версию.

---

## 🗂️ Создай рабочую папку

Храни код курса в одном месте, например:
```
D:\LearnProject\my-code\Python\
```
Открой её в VS Code через **Файл → Открыть папку**.

---

## 🧪 Бонус: запусти Python прямо в терминале (REPL)

Просто набери в PowerShell:
```powershell
python
```
Откроется интерактивный режим (REPL):
```
>>> 2 + 2
4
>>> print("Привет")
Привет
>>> exit()
```
Это удобно для быстрых проверок. Подробнее — в следующем блоке.
