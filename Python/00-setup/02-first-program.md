## 📖 Способ 1: REPL — интерактивный режим

**REPL** = Read-Eval-Print-Loop (читать → вычислить → напечатать → повторить). Это
«живой» режим, где Python отвечает сразу.

Запусти в терминале:
```powershell
python
```
И пробуй:
```python
>>> 2 + 2
4
>>> name = "Гена"
>>> print("Привет,", name)
Привет, Гена
>>> 10 * 5
50
>>> exit()        # выйти
```

💡 REPL идеален, чтобы быстро проверить идею или поэкспериментировать. Но программы из
многих строк удобнее писать в файле.

## 📖 Способ 2: файл с программой

Создай файл `hello.py` и напиши:

```python
print("Hello, world!")
```

Запусти из терминала (находясь в папке с файлом):
```powershell
python hello.py
```
Вывод:
```
Hello, world!
```

🎉 Твоя первая программа на Python!

> 💡 В VS Code можно просто нажать кнопку ▶️ в правом верхнем углу — она сделает то же самое.

## 📖 Разбираем программу побольше

```python
# Это комментарий — Python его игнорирует

name = input("Как тебя зовут? ")     # спросить имя у пользователя
age = int(input("Сколько тебе лет? "))   # спросить число (превратить текст в число)

print("Привет,", name + "!")
print("Через год тебе будет", age + 1)
```

| Строка | Что делает |
|--------|-----------|
| `# ...` | комментарий (для людей, не выполняется) |
| `input("...")` | показать текст и прочитать ввод пользователя (как строку) |
| `int(...)` | превратить строку в целое число |
| `print(...)` | вывести на экран |
| `name + "!"` | склеить строки |
| `age + 1` | арифметика с числом |

> ⚠️ `input()` **всегда** возвращает **строку** (текст). Чтобы считать число, оберни в
> `int()` или `float()`. Иначе `age + 1` сломается: нельзя прибавить число к тексту.
> Это частая ошибка новичка — запомни сразу.

## 🆚 Чем Python отличается визуально (после C)

Если ты пришёл из C — заметь различия:

```python
# Python                          // C
print("Hi")                       printf("Hi\n");      // нет ; и нет \n обязательно

if x > 0:                         if (x > 0) {
    print("плюс")                     printf("плюс");
                                  }
```

Главные отличия:
1. **Нет точек с запятой** `;` в конце строк.
2. **Нет фигурных скобок** `{}` — блоки задаются **отступами** (обычно 4 пробела)!
3. После `if`, `for`, `def` и т.п. ставится **двоеточие** `:`.
4. Не нужно объявлять тип переменной.

> ⚠️ **Отступы в Python — это синтаксис, а не оформление!** Неправильный отступ = ошибка
> `IndentationError`. Всегда используй **4 пробела** на уровень (VS Code делает это сам).

## 🧪 Эксперимент: посмотри байт-код

Помнишь из модуля 00, что Python превращает код в байт-код? Загляни:

```python
import dis
dis.dis("x = 1 + 2")
```

Ты увидишь внутренние инструкции (`LOAD_CONST`, `STORE_NAME`…). Пока их понимать не
нужно — но полезно знать, что твой код превращается **вот в это**.
