# 🚀 Пет-проект уровня 1 — Текстовая игра «Угадай число» + меню

> 🎯 **Цель:** собрать всё из уровня 1 (ввод-вывод, условия, циклы, функции) в одну
> законченную консольную игру.

---

## 📋 Что должна уметь программа

Игровое меню в цикле с несколькими мини-играми:

```
===== ИГРОВОЙ АВТОМАТ =====
1. Угадай число
2. Камень-ножницы-бумага
3. Викторина
4. Статистика
0. Выход
Выбор: 1

Загадал число от 1 до 100. Угадывай!
> 50
Больше!
> 75
Меньше!
> 62
🎉 Угадал за 3 попытки!
```

---

## ✅ Требования (минимум)

- [ ] Меню в цикле (`while`), выход по `0`.
- [ ] Минимум 2 мини-игры, каждая — **отдельная функция**.
- [ ] **Угадай число:** программа загадывает (`random.randint`), даёт подсказки
      «больше/меньше», считает попытки.
- [ ] Корректная обработка ввода (не падает на буквах вместо цифр).
- [ ] f-строки для красивого вывода.

---

## 🪜 Пошаговый план

1. **Меню.** Цикл `while True`, печать пунктов, чтение выбора, выход по `0`.
2. **Случайное число.** Подключи `import random`, используй `random.randint(1, 100)`.
3. **Угадай число.** Цикл угадывания с `break` при победе, счётчик попыток, подсказки.
4. **Вторая игра.** Камень-ножницы-бумага против компьютера (`random.choice`).
5. **Защита ввода.** Проверяй `isdigit()` или используй простую проверку.
6. **Статистика.** Считай сыгранные игры/победы в переменных, показывай в пункте меню.

## 🧱 Скелет

```python
import random

def guess_number():
    secret = random.randint(1, 100)
    attempts = 0
    print("Загадал число от 1 до 100!")
    while True:
        text = input("> ")
        if not text.isdigit():
            print("Введи число!")
            continue
        guess = int(text)
        attempts += 1
        if guess < secret:
            print("Больше!")
        elif guess > secret:
            print("Меньше!")
        else:
            print(f"🎉 Угадал за {attempts} попытки!")
            return attempts

def rock_paper_scissors():
    # реализуй сам: random.choice(["камень","ножницы","бумага"])
    pass

def print_menu():
    print("\n===== ИГРОВОЙ АВТОМАТ =====")
    print("1. Угадай число")
    print("2. Камень-ножницы-бумага")
    print("0. Выход")

def main():
    while True:
        print_menu()
        choice = input("Выбор: ")
        if choice == "0":
            print("Пока!")
            break
        elif choice == "1":
            guess_number()
        elif choice == "2":
            rock_paper_scissors()
        else:
            print("Нет такого пункта")

main()
```
