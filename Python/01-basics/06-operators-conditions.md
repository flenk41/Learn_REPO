# 06 · Операторы и условия

## 📖 Сравнение и логика

| Оператор | Значение |
|----------|----------|
| `==` | равно (по значению) |
| `!=` | не равно |
| `>` `<` `>=` `<=` | сравнения |
| `and` | И (оба истинны) |
| `or` | ИЛИ (хотя бы один) |
| `not` | НЕ (инверсия) |

```python
age = 25
print(age >= 18 and age < 65)    # True
print(age < 0 or age > 120)      # False
print(not (age == 25))           # False
```

> 💡 В отличие от C, в Python пишут словами `and`, `or`, `not`, а не `&&`, `||`, `!`.

### ⭐ is vs == — частая путаница

```python
a = [1, 2]
b = [1, 2]
print(a == b)    # True  — равны по ЗНАЧЕНИЮ (содержимое одинаково)
print(a is b)    # False — это РАЗНЫЕ объекты в памяти!
```

🖼️
```
   a ──► [объект-список #1: 1,2]
   b ──► [объект-список #2: 1,2]    содержимое одинаковое (==),
                                    но объекты разные (is → False)
```

💡 `==` сравнивает **значения**, `is` сравнивает **тождество объектов** (тот же объект в
памяти или нет). Это прямое продолжение темы про ссылки! Используй `==` для сравнения
значений, `is` — только с `None`: `if x is None:`.

## 📖 Условие `if / elif / else`

```python
score = 75

if score >= 90:
    print("Отлично")
elif score >= 70:
    print("Хорошо")
elif score >= 50:
    print("Удовлетворительно")
else:
    print("Неудовлетворительно")
```

> ⚠️ Не забывай **двоеточие** `:` после условия и **отступ** (4 пробела) для тела!

🖼️ Поток выполнения:
```
score >= 90 ? ──да──► "Отлично"
     │ нет
score >= 70 ? ──да──► "Хорошо"
     │ нет
     ▼ ... → else
```

## 📖 Что считается «истиной» и «ложью»

В условии можно использовать не только сравнения. «Ложными» считаются:

```python
# Falsy (ложь):
0, 0.0, "", [], {}, (), None, False

# Truthy (истина): всё остальное
42, "текст", [1], ...
```

```python
name = input("Имя: ")
if name:                      # пусто? → ложь
    print(f"Привет, {name}")
else:
    print("Ты не ввёл имя")

items = []
if not items:                 # пустой список → ложь
    print("Список пуст")
```

💡 Питоничный стиль: `if items:` вместо `if len(items) > 0:`.

## 📖 Тернарный оператор (короткий if)

```python
age = 20
status = "взрослый" if age >= 18 else "ребёнок"
print(status)
```
Читается как фраза: «взрослый, если age >= 18, иначе ребёнок».

---

## 📖 Вложенные условия и цепочки сравнений

```python
# Python умеет цепочки — так нельзя в C!
if 0 <= age <= 120:
    print("корректный возраст")

# вложенность
if logged_in:
    if is_admin:
        print("админ-панель")
    else:
        print("обычный пользователь")
```

> 💡 `0 <= age <= 120` — питоническая цепочка, эквивалент `age >= 0 and age <= 120`.

## 📖 match — выбор из вариантов (Python 3.10+)

Аналог `switch` из C:

```python
command = input("Команда: ")

match command:
    case "старт":
        print("Запуск")
    case "стоп":
        print("Остановка")
    case "выход" | "quit":      # несколько вариантов
        print("Пока")
    case _:                     # _ = всё остальное (default)
        print("Неизвестная команда")
```

## ✅ Задачи

1. **Чёт/нечет.** Считай число, скажи чётное оно или нечётное.
2. **Максимум из трёх.** Считай три числа, выведи наибольшее (без `max()`).
3. **Калькулятор.** Считай два числа и операцию (`+ - * /`), выведи результат через
   `if/elif` или `match`. Обработай деление на ноль.
4. **Високосный год.** Год високосный, если делится на 4, но не на 100, либо на 400.
5. **Оценка.** Считай балл 0–100, выведи букву A/B/C/D/F.
6. **Знак числа.** Через тернарный оператор выведи `-1`, `0` или `1`.
7. **Категория ИМТ.** Считай вес и рост, посчитай ИМТ, выведи категорию.
8. **is vs ==.** Создай два одинаковых списка, проверь `==` и `is`. Объясни результат.
