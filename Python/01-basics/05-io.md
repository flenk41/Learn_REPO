# 05 · Ввод и вывод

> 🎯 **Цель блока:** уверенно общаться с пользователем — выводить данные и считывать ввод.

---

## 📖 Вывод: `print`

```python
print("Привет")
print("a", "b", "c")            # Привет → a b c  (через пробел)
print("a", "b", sep="-")        # a-b  (свой разделитель)
print("без переноса", end="")   # не переходить на новую строку
print(1, 2, 3, sep=", ")        # 1, 2, 3
```

| Параметр | Что делает |
|----------|-----------|
| `sep="..."` | разделитель между значениями (по умолчанию пробел) |
| `end="..."` | что напечатать в конце (по умолчанию `\n` — перенос) |

### Форматированный вывод через f-строки

```python
name = "Гена"
score = 95.5
print(f"{name}: {score} баллов")
print(f"Цена: {1234.5:.2f} руб")     # 1234.50 — два знака
print(f"{42:>5}")                    # "   42" — выравнивание вправо по ширине 5
print(f"{0.25:.0%}")                 # "25%" — проценты
```

## 📖 Ввод: `input`

```python
name = input("Как тебя зовут? ")     # показать текст и прочитать ответ
print(f"Привет, {name}!")
```

### ⭐ Главное про input: он ВСЕГДА возвращает строку

```python
age = input("Возраст: ")    # "25" — это СТРОКА, даже если ввели число
print(age + 1)              # ⚠️ ОШИБКА! нельзя строка + число
```

🖼️
```
   пользователь ввёл:  25
   input() вернул:    "25"  ← строка (текст), а не число!
```

Чтобы получить число — **преврати** строку:

```python
age = int(input("Возраст: "))       # теперь это int
price = float(input("Цена: "))      # теперь это float
print(age + 1)                      # ✅ работает
```

> ⚠️ Это ошибка №1 новичков в Python. Запомни: **число из input всегда оборачивай в
> `int()` или `float()`**.

---

## 📖 Считать несколько значений в одной строке

```python
# пользователь вводит: 3 5
a, b = input("Два числа: ").split()    # split() режет по пробелам → ['3', '5']
a, b = int(a), int(b)
print(a + b)

# или короче — map применяет int к каждому куску
a, b = map(int, input("Два числа: ").split())
print(a + b)
```

🖼️
```
   "3 5"  --split()-->  ['3', '5']  --map(int)-->  3, 5
```

## ⚠️ Защита от неправильного ввода

Если пользователь введёт «abc» вместо числа — `int()` упадёт с ошибкой. Профи это
обрабатывают (подробно об исключениях — в Уровне 3):

```python
text = input("Число: ")
if text.isdigit():
    n = int(text)
    print(n * 2)
else:
    print("Это не число!")
```
