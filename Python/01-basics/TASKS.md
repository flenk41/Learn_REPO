# ✅ Задачи · Уровень 1 (основы Python)

> Решай **сам**. Застрял на 30+ минут — подсмотри идею, но код пиши руками.
> Сложность: ⭐ легко · ⭐⭐ средне · ⭐⭐⭐ сложно

---

## 🟢 Блок A — переменные, числа, строки

1. ⭐ **Обмен.** Считай две переменные, поменяй местами через `a, b = b, a`.
2. ⭐ **Конвертер.** Километры → мили (× 0.621), вывод с 2 знаками.
3. ⭐ **Инициалы.** Из «Фамилия Имя Отчество» сделай «Фамилия И.О.».
4. ⭐⭐ **Зеркало.** Считай слово, проверь, палиндром ли оно (через срез `[::-1]`).
5. ⭐⭐ **Шифр сдвига.** Сдвинь каждую букву строки на 1 (без спецбиблиотек, через `ord`/`chr`).

## 🟢 Блок B — условия

6. ⭐ **Время суток** по введённому часу (ночь/утро/день/вечер).

8. ⭐⭐ **Квадратное уравнение.** Реши `ax²+bx+c=0`, обработай все случаи.
9. ⭐⭐ **Камень-ножницы-бумага** для двух игроков.
10. ⭐⭐ **Калькулятор ИМТ** с категориями.
