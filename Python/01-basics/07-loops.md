## 🔁 Цикл `for` — перебор последовательности

В Python `for` перебирает **элементы** чего-либо (а не «считает», как в C):

```python
for letter in "Привет":
    print(letter)            # печатает каждую букву

for item in ["яблоко", "груша", "слива"]:
    print(item)              # печатает каждый элемент списка
```

🖼️
```
   for x in [10, 20, 30]:
       x по очереди = 10, потом 20, потом 30
```

### `range()` — генератор чисел

Чтобы повторить N раз или пройти по числам:

```python
for i in range(5):           # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):        # 1, 2, 3, 4, 5  (от 1 до 6, не включая 6)
    print(i)

for i in range(0, 10, 2):    # 0, 2, 4, 6, 8  (шаг 2)
    print(i)

for i in range(5, 0, -1):    # 5, 4, 3, 2, 1  (обратный отсчёт)
    print(i)
```

| Вызов | Числа |
|-------|-------|
| `range(n)` | 0 … n−1 |
| `range(a, b)` | a … b−1 |
| `range(a, b, step)` | a, a+step, … (до b) |

> 💡 `range` не создаёт весь список чисел в памяти — он генерирует их по одному (ленивость).
> Это важная для памяти идея, которую глубоко разберём в Уровне 3 (генераторы).

### `enumerate` — индекс + значение

```python
fruits = ["яблоко", "груша", "слива"]
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")       # 0: яблоко, 1: груша, ...
```

## 🔁 Цикл `while` — пока условие истинно

```python
count = 0
while count < 5:
    print(count)
    count += 1            # ⚠️ без этого — бесконечный цикл!
```

> ⚠️ Если забыть менять переменную условия — цикл крутится вечно. Останови: **Ctrl + C**.

Частый приём — цикл «до правильного ввода»:
```python
while True:
    text = input("Введи число > 0: ")
    if text.isdigit() and int(text) > 0:
        break            # выйти, когда ввод корректный
    print("Неверно, попробуй ещё")
```

## 🎛️ Управление: `break`, `continue`, `else`

```python
for i in range(10):
    if i == 5:
        break            # полностью выйти из цикла
    print(i)             # 0 1 2 3 4

for i in range(10):
    if i % 2 == 0:
        continue         # пропустить чётные
    print(i)             # 1 3 5 7 9
```

### `else` у цикла — редкая, но полезная фишка

```python
for x in [1, 3, 5]:
    if x % 2 == 0:
        print("нашли чётное")
        break
else:
    print("чётных нет")   # выполнится, если цикл прошёл БЕЗ break
```

## 📖 Вложенные циклы

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i}×{j}={i*j}", end="  ")
    print()              # перенос строки после каждой строки таблицы
```

## ✅ Задачи

1. **Счёт.** Выведи числа от 1 до 20 в одну строку.
2. **Сумма.** Посчитай сумму чисел от 1 до 100 (= 5050).
3. **Факториал.** Считай `n`, посчитай `n!`.
4. **Таблица умножения** 9×9 вложенными циклами.
5. **Реверс числа.** Считай число, выведи задом наперёд (123 → 321), без срезов строк.
6. **Простое число?** Определи, простое ли введённое число.
7. **Сумма цифр.** Посчитай сумму цифр числа.
8. **Угадай число.** Загадай число (например 42), пользователь угадывает, программа
   говорит «больше»/«меньше», считает попытки (`while True` + `break`).
9. **Ёлочка.** Считай высоту `n`, нарисуй пирамиду из звёздочек (вложенные циклы или
   `"*" * k`).
10. **FizzBuzz.** Числа 1–100: кратные 3 → «Fizz», кратные 5 → «Buzz», кратные обоим →
    «FizzBuzz», иначе само число.
