# 01 · Установка Roblox Studio 🖼️

> 🎯 **Цель блока:** поставить Studio, создать первое место (place) и запустить его — чтобы дальше
> всё пробовать руками.

---

## 🛠️ Установка

```
   1. заведи аккаунт на roblox.com (если нет). запомни логин/пароль.
   2. скачай Roblox Studio: create.roblox.com → «Start Creating» / «Download Studio». Бесплатно.
      Windows и macOS. (Studio — отдельная программа, не путать с игровым клиентом Roblox.)
   3. установи, запусти, войди своим аккаунтом.
```

💡 Studio бесплатна и не требует мощного ПК для простых игр. Один аккаунт Roblox — и для игры, и для
создания. Всё, что ты создаёшь, привязано к твоему аккаунту.

---

## ⭐ Первое место (place)

```
   в Studio при запуске — окно шаблонов (Templates):
   • Baseplate — пустая платформа. ЛУЧШИЙ старт для обучения (чистый лист).
   • Classic Baseplate / Flat Terrain — варианты пустых сцен.
   • готовые шаблоны (Obby, Racing, ...) — каркасы игр (потом, для вдохновения).

   выбери BASEPLATE → откроется 3D-сцена с серой платформой и кнопкой ▶ Play.

   термины:
   • PLACE — конкретная сцена/уровень (один файл игры).
   • EXPERIENCE (game) — твоя игра целиком (может содержать несколько places).
```

💡 ⭐ Начинай с **Baseplate** — пустая сцена, ничего не отвлекает. Сохраняй часто: File → Save to
File (локально, .rbxl) на время обучения; позже — публикация в Roblox (модуль 22).

---

## ⭐ Запуск и тест

```
   ▶ PLAY (или F5)   — запустить игру: ты появляешься персонажем в своём мире, можно ходить/прыгать.
   ⏹ Stop (Shift+F5) — остановить тест, вернуться к редактированию.
   режимы теста (вкладка Test): Play, Play Here, Run (без персонажа), Start (мультиплеер — несколько окон).

   ⚠️ важно: изменения, сделанные ВО ВРЕМЯ Play, НЕ сохраняются (это тестовый прогон).
   правь в режиме редактирования (когда Stop), тестируй в Play.
```

💡 ⭐ Цикл разработки: правишь сцену/скрипт (Edit) → ▶ Play (проверяешь) → Stop → правишь дальше.
Привыкни смотреть панель **Output** (View → Output) — там печатаются ошибки скриптов и `print()`
(модуль 08). Это твой «терминал».

---

## 📖 Если Studio не ставится / тормозит

```
   • проверь интернет (Studio тянет ассеты и проверяет вход).
   • перезапусти/переустанови; обнови ОС/драйверы видеокарты.
   • на слабом ПК: упрости графику (File → Studio Settings → Rendering → quality пониже).
   • вошёл не тем аккаунтом → выйди и войди нужным.
```

---

## ⚠️ Ловушки

- ❌ Скачать игровой клиент Roblox вместо Studio (это разные программы).
- ❌ Править мир во время ▶ Play и удивляться, что изменения пропали (Play — тест, не сохраняется).
- ❌ Не открыть Output → не видеть ошибок скриптов (потом «ничего не работает» без причины).
- ❌ Забывать сохранять (Ctrl+S / Save to File).

---

## ✅ Задачи

1. Поставь Studio, войди аккаунтом, создай место по шаблону **Baseplate**.
2. Нажми ▶ Play — походи персонажем. Нажми Stop.
3. Открой панель **Output** (View → Output). Пока пусто — скоро пригодится.
4. Сохрани место в файл (File → Save to File). Найди .rbxl на диске.
5. ⭐ Попробуй режим **Start** с 2 игроками (Test → Clients and Servers) — увидишь мультиплеер.
