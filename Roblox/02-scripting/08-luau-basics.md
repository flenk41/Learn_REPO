## 📖 Где живут скрипты

```
   СКРИПТ — Instance с кодом на Luau. виды (модуль 11 подробнее):
   • Script        — выполняется на СЕРВЕРЕ. кладут в ServerScriptService.
   • LocalScript   — на КЛИЕНТЕ (у игрока). кладут в StarterPlayerScripts / StarterGui.
   • ModuleScript  — переиспользуемый код (библиотека), сам не запускается — его require'ят.

   создать: ПКМ на контейнере (напр. ServerScriptService) → Insert Object → Script.
   запуск: ▶ Play. вывод — в Output (View → Output).
```

## ⭐ Основы синтаксиса

```lua
   -- комментарий (два дефиса)

   print("Привет, Roblox!")          -- вывод в Output

   local score = 0                   -- local — локальная переменная (всегда используй local!)
   local name = "Игрок"
   local isAlive = true
   local pi = 3.14

   score = score + 10                -- присваивание

   -- типы значений: number, string, boolean, nil, table, и Instance (объекты Roblox)
   -- строки: .. — склейка
   print("Очки: " .. score)          -- Очки: 10
```

💡 ⭐ Всегда пиши **`local`** перед переменной (без него она глобальная — медленнее и source багов).
`print()` + панель **Output** — твой главный инструмент отладки: печатай значения, смотри, что
происходит. Склейка строк — через `..` (не `+`).

## ⭐⭐ Условия, циклы, функции

```lua
   -- УСЛОВИЕ
   if score >= 100 then
       print("Уровень пройден!")
   elseif score >= 50 then
       print("Почти")
   else
       print("Собирай дальше")
   end

   -- ЦИКЛЫ
   for i = 1, 5 do                    -- от 1 до 5
       print("шаг " .. i)
   end

   while score < 100 do
       score = score + 10
   end

   -- ФУНКЦИЯ
   local function addScore(current, amount)
       return current + amount
   end
   score = addScore(score, 25)       -- вызов
```

💡 ⭐⭐ Блоки в Luau закрываются словом **`end`** (не скобки/отступы): `if...end`, `for...end`,
`function...end`. Это главное отличие от C/Python синтаксиса. Функции группируют логику — как и в
любом языке (`local function имя(параметры) ... end`).

## ⭐ Таблицы (table) — главная структура

```lua
   -- table — универсальная структура: и список, и словарь.
   local fruits = {"яблоко", "банан", "вишня"}   -- список (индексы с 1!)
   print(fruits[1])                              -- яблоко  (Luau нумерует с 1, не с 0!)

   local player = {                              -- словарь (ключ-значение)
       name = "Аня",
       score = 0,
       level = 1,
   }
   print(player.name)                            -- Аня
   player.score = 50

   -- перебор
   for index, value in ipairs(fruits) do print(index, value) end   -- список
   for key, value in pairs(player) do print(key, value) end        -- словарь
```

💡 ⭐ Таблица — единственная структура данных Luau (и список, и словарь, и объект). Запомни две
ловушки новичков из других языков: **индексы с 1** (не с 0!) и блоки через **`end`**.

## 📖 Типы Luau (опционально, но полезно)

```lua
   -- Luau поддерживает АННОТАЦИИ ТИПОВ (в отличие от ванильного Lua):
   local count: number = 0
   local function greet(name: string): string
       return "Привет, " .. name
   end
   -- типы помогают ловить ошибки до запуска (Studio подсветит). для больших проектов — полезно.
```

## ⚠️ Ловушки

- ❌ Забыть `local` → глобальная переменная (медленно, конфликты, баги).
- ❌ Индексировать список с 0 (в Luau — с 1!).
- ❌ Скобки/отступы вместо `end` для блоков.
- ❌ Склеивать строки через `+` вместо `..`.
- ❌ Не открыть Output → не видеть `print`/ошибок (работа вслепую).
- ❌ Сравнение `=` vs `==` (присваивание vs сравнение).
