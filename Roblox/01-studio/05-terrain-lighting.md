# 05 · Terrain, освещение, атмосфера 🖼️

> 🎯 **Цель блока:** сделать мир красивым и живым — ландшафт (Terrain), освещение (Lighting) и
> атмосферные эффекты. Атмосфера сильно влияет на впечатление от игры.

---

## ⭐ Terrain — ландшафт

```
   TERRAIN — отдельная система для природного рельефа (холмы, вода, пещеры) — не из Parts.
   редактор: вкладка Terrain Editor (Create / Edit / Region).
   • Create — кисти: добавить землю, траву, камень, песок, воду.
   • Edit — Sculpt (лепить), Smooth (сглаживать), Erode (размывать), Paint (красить материалом).
   • вода (Water) — особый материал: течёт, в ней плавают.

   Terrain хорош для открытых миров; для платформ/строений — обычные Parts.
```

💡 ⭐ Terrain vs Parts: **Parts** — для рукотворного (платформы, здания, механизмы), **Terrain** —
для природного (земля, горы, вода). Для симулятора часто хватает Parts + немного Terrain для фона.
Не увлекайся тяжёлым Terrain на старте — он влияет на производительность (модуль 20).

---

## ⭐⭐ Lighting — освещение и время суток

```
   LIGHTING (контейнер в Explorer) — глобальное освещение сцены:
   • ClockTime / TimeOfDay — время суток (0-24), положение солнца.
   • Brightness — яркость солнца.   • Ambient / OutdoorAmbient — заполняющий свет (тени не чёрные).
   • FogEnd / атмосфера — дальность видимости/туман.

   ИСТОЧНИКИ света (Instances внутри Part — локальный свет):
   • PointLight — точечный (лампочка).   • SpotLight — прожектор (конус).   • SurfaceLight — от грани.

   ⚠️ Neon-материал светится сам, но НЕ освещает округу — для реального света добавь PointLight.
```

💡 ⭐⭐ Освещение задаёт настроение: ночь + точечные огни = атмосфера; яркий день = казуальная игра.
Меняй ClockTime, Ambient, добавляй PointLight в фонари. Это дешёвый способ сделать мир «дорогим» на
вид. Глобальный свет — в Lighting; локальные лампы — как дети Parts.

---

## 📖 Атмосферные эффекты

```
   добавляются как Instances в Lighting (ПКМ на Lighting → Insert):
   • Atmosphere — реалистичная дымка/глубина (современный способ тумана).
   • Sky — кастомное небо (скайбокс): свои текстуры неба/звёзд.
   • Bloom, SunRays, DepthOfField, ColorCorrection — пост-эффекты (свечение, лучи, размытие, цвет).

   немного эффектов сильно улучшают картинку. но не перегружай — бьёт по производительности.
```

---

## ⚠️ Ловушки

- ❌ Тяжёлый Terrain везде → тормоза (модуль 20). Используй по делу.
- ❌ Ждать, что Neon осветит сцену (он только светится сам; нужен PointLight/SpotLight).
- ❌ Слишком тёмный Ambient → ничего не видно в тенях.
- ❌ Перебор пост-эффектов (Bloom/DOF на максимум) → каша и просадки FPS.
- ❌ Путать глобальный свет (Lighting) и локальные источники (внутри Parts).
