# ✅ Задачи уровня 2 · SQL и запросы

> Ядро трека. SQL осваивается ТОЛЬКО за клавиатурой. Используй учебную базу (dvdrental/chinook) или
> свою из Уровня 1. Пиши десятки запросов.

---

## 🟢 Базовые

1. **SELECT + WHERE.** Выборки с разными фильтрами (AND/OR/IN/BETWEEN/LIKE/IS NULL). Сортировка, LIMIT.
2. **INNER JOIN.** Соедини две таблицы по связи (заказы + клиенты, фильмы + категории).
3. **Агрегаты.** COUNT/SUM/AVG по таблице; GROUP BY с итогами по группам.

## 🟡 Средние

4. **LEFT JOIN.** Все строки левой + связанные; найди записи БЕЗ связанных (IS NULL).
5. **N:M через JOIN.** Соедини через связующую таблицу (заказы→товары, фильмы→актёры).
6. **GROUP BY + HAVING.** Группы с фильтром по агрегату (клиенты с >N заказами). Топ-10 по сумме (JOIN+GROUP BY).
7. **Подзапрос.** «Выше среднего», IN-подзапрос, EXISTS.

## 🔴 Со звёздочкой

8. ⭐ **CTE.** Перепиши сложный вложенный запрос через WITH (несколько шагов). Сравни читаемость.
9. ⭐ **Аналитический отчёт.** Сложный запрос: JOIN нескольких таблиц + GROUP BY + HAVING + ORDER BY +
   вычисления (например, «выручка по категориям по месяцам, топ-5»).
10. ⭐ **Оконные функции.** Ранг/топ-N в группе (RANK/ROW_NUMBER OVER PARTITION BY), нарастающий итог.
11. ⭐ **CRUD + транзакция.** INSERT/UPDATE/DELETE; перевод между «счетами» в транзакции с ROLLBACK.
