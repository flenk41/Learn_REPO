# 🧠 Курс «Работа с памятью» — от новичка до Senior

Курс, который проведёт тебя по пути **от новичка до Senior**. Главная цель —
по-настоящему понять, **как работает память**: где живут переменные, что
такое стек и куча, как работают указатели, откуда берутся утечки и как писать
быстрый, безопасный и предсказуемый код. А в треке по **ИИ** ты увидишь, что у моделей
тоже есть память — **контекст** — и научишься ею управлять.

> Вдохновлён форматом дорожных карт [roadmap.sh](https://roadmap.sh/cpp), но с акцентом
> на память, практику и пет-проекты после каждого блока.

---

## 💻 Веб-интерфейс (читать как сайт)

Курс можно проходить прямо в браузере — с навигацией по уровням, **сохранением
прогресса**, подсветкой кода, схемами памяти и интерактивной дорожной картой.

**Как запустить (Windows):**

1. Дважды кликни на **`start-server.bat`** в папке `D:\LearnProject`.
2. Откроется браузер на `http://localhost:8000` — всё готово.
3. Вверху панели слева переключай трек: **🇨 C** · **➕ C++** · **🐍 Python** · **🦀 Rust** · **🤖 ИИ** · **🔗 Интеграция** · **⚡ Claude Code** · **🌐 Сети** · **🖥️ ОС** (прогресс хранится отдельно).

> Файл сам найдёт Python или Node.js и поднимет локальный сервер. Если ничего не
> установлено — поставь [Python](https://python.org) (галочка «Add to PATH») и запусти снова.

**Вручную**, если хочешь сам:
```powershell
cd D:\LearnProject
py -m http.server 8000
# затем открой http://localhost:8000
```

> ⚠️ Просто открыть `index.html` двойным кликом **не получится** — браузер блокирует
> чтение файлов через `file://`. Нужен локальный сервер (см. выше).

Предпочитаешь читать прямо в редакторе/на GitHub? Все материалы — обычный markdown,
начни с [дорожной карты C](C/README.md).

## 📚 Языки курса

| № | Язык | Статус | Зачем он в курсе |
|---|------|--------|------------------|
| 1 | **C** | ✅ готов | Базовый язык. Здесь нет «магии» — ты руками управляешь памятью (malloc/free) и видишь, как всё устроено внизу. |
| 2 | **C++** | ✅ готов | Золотая середина: ручной контроль **плюс** безопасность — RAII, умные указатели, move. Автоочистка без сборщика мусора. |
| 3 | **Python** | ✅ готов | Управляемая память: имена-ярлыки, объекты, подсчёт ссылок, сборщик мусора — взгляд «сверху». |
| 4 | **Rust** | ✅ готов | Кульминация: безопасность памяти **без сборщика мусора**, проверенная компилятором (владение + borrow checker). |
| 5 | **🤖 ИИ** | ✅ готов | Бонус-трек: промпты и работа с ИИ. Ядро — **контекст** (память модели): тот же принцип памяти, только у ИИ. |
| 6 | **🔗 Интеграция** | ✅ готов | Вершина: как связать языки (C+Python, Rust+Python, WASM) и писать **драйверы**. Ядро — **память на границе** языков и память железа. |
| 7 | **⚡ Claude Code** | ✅ готов | Инструмент-трек: как превратить Claude Code в рабочую среду агента (MCP, Skills, саб-агенты, hooks). Ядро — **контекстное окно** (память агента): тот же принцип памяти, в работе с ИИ-инструментом. |
| 8 | **🌐 Сети** | ✅ готов | Как работает интернет: IP, DNS, маршрутизация, HTTP/TLS, облако. Ядро — **транспорт (TCP/UDP)**: как данные путешествуют между программами. Байты те же, что в памяти, только в пути по сети. |
| 9 | **🖥️ ОС** | ✅ готов | Как работает операционная система: процессы, потоки, синхронизация, контейнеры. Ядро — **виртуальная память**: кто создаёт и раздаёт ту память, которой ты управлял в языках. Самая глубокая связь с темой памяти. |

Четыре языка дают **полную картину памяти**: в C ты управляешь ею **вручную**, в C++ —
**вручную, но с инструментами безопасности** (RAII), в Python — **автоматически** (GC), а в
Rust — **безопасно без GC**, и компилятор это доказывает. А трек по **ИИ** показывает, что
память есть и у моделей — это **контекстное окно**, и им тоже нужно уметь управлять.
Понимание памяти на всех уровнях — от байтов в C до контекста ИИ — это и есть глубокое
понимание.

Можно проходить любой первым. Хочешь понять память «от земли» — начни с C. Контроль +
безопасность — C++. Быстро писать работающее — Python. Безопасность без компромиссов — Rust.
Выжать максимум из ИИ — трек **🤖 ИИ** (программирование не нужно). А когда освоишь языки —
**🔗 Интеграция** научит связывать их вместе и писать системный код.

## 🎯 Как устроен курс

Каждый **блок** построен одинаково:

1. **📖 Теория** — простым языком, с картинками и схемами памяти.
2. **🛠️ Практика** — как установить, настроить, запустить.
3. **✅ Задачи** — закрепить материал (от простых к сложным).
4. **🚀 Пет-проект** — собрать всё изученное в один реальный проект.
5. **🔍 Проверка себя** — вопросы, на которые ты должен уметь ответить.

## 📈 Уровни прогресса

```
🥚 Новичок  →  🐣 Junior  →  🐥 Middle  →  🦅 Senior
   (основы)    (синтаксис)   (структуры)   (системное
                              данных,        программирование,
                              память)        память «по-взрослому»)
```

| Уровень | Что умеешь |
|---------|-----------|
| 🥚 **Новичок** | Установил компилятор, запустил первую программу, понимаешь что такое байт. |
| 🐣 **Junior** | Пишешь программы с условиями, циклами, функциями. Понимаешь типы и их размер. |
| 🐥 **Middle** | Свободно работаешь с указателями, динамической памятью, структурами данных. |
| 🦅 **Senior** | Пишешь свои аллокаторы, понимаешь UB, выравнивание, кэш, многопоточность, оптимизацию. |
