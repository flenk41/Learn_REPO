# 🚀 Проект уровня 1 · «Детектор красных флагов»

> Собери практический инструмент защиты — справочник-памятку по распознаванию атак соц. инженерии,
> которым реально будешь пользоваться сам и сможешь дать близким/коллегам.

> ⚠️ Проект — **оборонительный**: про распознавание и защиту. Никаких реальных атак на людей.

---

## 🎯 Что делаем

Создай «детектор красных флагов» — структурированную памятку/чек-лист, помогающую за секунды
оценить подозрительное письмо, звонок или сообщение. Формат на выбор: markdown-документ,
одностраничная карточка, простая веб-страница, заметка в телефоне.

```
   подозрительное сообщение → прогон по чек-листу → вердикт: доверять / проверить / удалить
```

---

## 📋 Что включить

### 1. Чек-лист красных флагов (по каналам)
- **Почта (фишинг):** отправитель, срочность/угроза, ссылки, вложения, просьба, мелочи.
- **Звонок (вишинг):** просят секреты/действие, давят, «не кладите трубку», подмена номера.
- **SMS/мессенджер (смишинг):** ссылки, «посылка/приз/SOS от близкого», просьба кода.
- **Личное общение (претекстинг/физ.):** непроверяемая роль, обход процедур, «на хвосте».

### 2. Универсальные правила-рефлексы
- Торопят/пугают → **пауза**, не действуй в моменте.
- Просят пароль/код/перевод → **стоп**, легитимные так не делают.
- Проверяй через **официальный/независимый канал**, а не из самого сообщения.
- Сомневаешься → **не делай**, переспроси/сообщи.

### 3. Мои официальные каналы
- Таблица: организация → официальный способ связи (телефон/приложение/сайт), которым перепроверяешь.

### 4. Что делать при подозрении
- Куда сообщить (на работе — в IT/безопасность; лично — в банк/поддержку), что НЕ делать (не
  кликать, не отвечать данными, не удалять молча на работе).

---

## 🌟 Бонусы

- 🥚 Сделай отдельную **версию для нетехнических близких** — простым языком, крупно, с примерами.
- 🐣 Добавь 5 реальных примеров (обезличенных) подозрительных сообщений с подсветкой флагов.
- 🦅 Оформи как интерактивную страничку: вводишь признаки → получаешь «уровень подозрительности».
- 🚀 Договорись с семьёй о «кодовом слове» и правиле перепроверки для «SOS, срочно нужны деньги».

---

## ✅ Критерии готовности

- [ ] Чек-листы по 4 каналам (почта/звонок/SMS/личное) с конкретными флагами
- [ ] Универсальные правила-рефлексы (пауза, стоп, независимый канал)
- [ ] Таблица «мои официальные каналы» для перепроверки
- [ ] Раздел «что делать при подозрении» (куда сообщить, чего не делать)
- [ ] Версия/вариант, понятный нетехническому человеку
